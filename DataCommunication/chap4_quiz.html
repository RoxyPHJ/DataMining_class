단답형 ⭐⭐ Slide 5
회선 부호화의 3가지 문제점을 모두 쓰시오.
✅ 정답
  1. 기준선 표류(Baseline Wandering)
  2. 직류 성분(DC Component)
  3. 자기 동기화(Self-Synchronization)
해설
① 기준선 표류: 수신 신호의 평균을 기준으로 데이터를 결정하는데, 연속된 0 또는 1이 지속되면 기준선이 이동하여 복호화 오류 발생.
② 직류 성분: 저주파(DC) 성분은 전선·변압기를 통과하지 못하므로 제거 필요.
③ 자기 동기화: 송·수신자의 비트 간격이 일치하지 않으면 복호화 오류 발생.

계산형 ⭐⭐ Slide 4
데이터 전송률(N)이 4,000 bps이고, 하나의 신호 요소가 2개의 데이터 요소를 나타낼 때, 신호 전송률(S)은 얼마인가? (단위 포함)
✅ 정답
2,000 baud
풀이
공식: S = N / r
r = 2 (신호 요소 1개당 데이터 요소 2개)
S = 4,000 / 2 = 2,000 baud
r이 2이므로, 하나의 신호가 2비트를 표현 → 신호 전송률은 데이터 전송률의 절반이 된다.

단답형 ⭐⭐ Slide 13–14
나이퀴스트 정리(Nyquist Theorem)에 따르면, 최대 주파수가 4,000 Hz인 전화 음성 신호를 디지털화할 때 필요한 최소 표본화율(Sampling Rate)은 얼마인가? 또한 이 조건이 왜 필요한지 간략히 설명하시오.
✅ 정답
8,000 samples/sec (8,000 Hz)
풀이 및 해설
나이퀴스트 정리: fs ≥ 2 × fmax
fs = 2 × 4,000 = 8,000 samples/sec
표본화율이 최대 주파수의 2배 미만이면 에일리어싱(Aliasing)이 발생하여 원래 신호를 정확히 복원할 수 없기 때문에 이 조건이 필요하다.

계산형 ⭐⭐ Slide 17–18
인간의 목소리(0~4,000 Hz)를 디지털화할 때, 표본당 8 bits를 사용한다면 PCM 비트율(Bit Rate)은 얼마인가? (강의 예제 4.14)
✅ 정답
64,000 bps (64 Kbps)
풀이
Bit Rate = fs × nb
fs = 2 × 4,000 = 8,000 samples/sec
Bit Rate = 8,000 × 8 = 64,000 bps = 64 Kbps
이 값(64 Kbps)이 전화 음성 채널 1개의 표준 디지털 대역폭이다.

해설

계산형 [응용] 역방향 ⭐⭐ Slide 4
어떤 시스템에서 데이터 전송률(N)이 12,000 bps이고, 신호 전송률(S)이 4,000 baud로 측정되었다. 이 시스템에서 변수 r (신호 요소당 데이터 요소 수)은 얼마인가? 또한 이것이 의미하는 바를 설명하시오.
✅ 정답
r = 3
풀이 과정
공식 S = N / r 을 r에 대해 역산:
r = N / S = 12,000 / 4,000 = 3
의미: r = 3이란 하나의 신호 요소가 3개의 데이터 비트를 나타낸다는 뜻이다. 예를 들어 8준위(2³) 신호 방식이 이에 해당한다. r이 클수록 동일한 대역폭으로 더 많은 데이터를 전송할 수 있어 효율적이다.

계산형 [응용] 수치 변형 ⭐⭐ Slide 13–14
의료용 초음파 장비는 최대 주파수 22,000 Hz의 신호를 디지털화해야 한다. 나이퀴스트 정리를 적용하여 필요한 최소 표본화율을 구하고, 전화 음성(4,000 Hz) 대비 몇 배인지 계산하시오.
✅ 정답
44,000 samples/sec (44,000 Hz) — 전화 음성 대비 5.5배

참고로 CD 음질 표준(44,100 Hz)이 이 값과 유사한 이유가 바로 나이퀴스트 정리에 의한 것이다 (가청 주파수 최대 약 20,000 Hz).

맨체스터(Manchester)와 차분 맨체스터(Differential Manchester) 부호화의 공통점 1가지와 차이점 2가지를 서술하시오.
✅ 모범 답안
공통점
비트 간격 중간에서 반드시 전환(반전)이 발생하여 자기 동기화를 달성하고, DC 성분이 없다.

계산형 [응용] 수치 변형 ⭐⭐ Slide 17–18
고음질 음악 스트리밍 서비스에서 최대 주파수 20,000 Hz의 오디오 신호를 표본당 16 bits로 디지털화하고, 2채널(스테레오)로 전송한다면 총 비트율(Bit Rate)은 얼마인가?
✅ 정답
1,280,000 bps = 1.28 Mbps
풀이 과정
Step 1: fs = 2 × 20,000 = 40,000 samples/sec
Step 2: 1채널 Bit Rate = 40,000 × 16 = 640,000 bps
Step 3: 스테레오(2채널) = 640,000 × 2 = 1,280,000 bps = 1.28 Mbps
개념 적용: 표본화율은 나이퀴스트 정리(fs≥2×fmax)를 적용하고, PCM 공식(Bit Rate = fs × nb)을 채널 수만큼 곱한다. CD 음질(44,100 Hz × 16bit × 2ch = 1.4 Mbps)과 유사한 값이다.

OX 문제 [응용] 조건 변형 ⭐⭐⭐ Slide 15
양자화 구간 수(L)를 8에서 16으로 늘리면, 양자화 잡음(Quantization Noise)은 감소하지만 부호화에 필요한 비트 수(nb)도 반드시 증가한다.
O
X
✅ 정답
O (참)
해설
양자화 잡음 감소: L이 클수록 구간 높이 Δ가 작아져 근사 오차(잡음) 감소 → ✅ 참
비트 수 증가: L개 구간을 표현하려면 nb = log₂L 비트가 필요.
L=8 → nb = log₂8 = 3 bits
L=16 → nb = log₂16 = 4 bits
따라서 L을 늘리면 잡음은 줄지만 비트율(Bit Rate)이 증가하는 트레이드오프가 발생한다. 두 명제 모두 참이므로 전체 문장은 O이다.

객관식 [응용] 시나리오 분석 ⭐⭐⭐ Slide 5, 7–8
NRZ-L 방식으로 데이터를 전송할 때, 수신기에서 복호화 오류가 발생할 가능성이 가장 높은 비트열은? (기준선 표류 관점에서 판단)
A 1 0 1 0 1 0 1 0 (교번 패턴)
B 1 1 1 1 1 1 1 1 (연속 1)
C 1 0 0 1 1 0 1 0 (혼합)
D 1 1 0 0 1 1 0 0 (2개씩 반복)
✅ 정답
B — 1 1 1 1 1 1 1 1 (연속 1)
해설
기준선 표류는 지속적인 0 또는 1이 전송될 때 발생한다. NRZ-L에서 연속 1은 계속 음(−V) 전압이 유지되어 수신기의 평균 기준선이 음의 방향으로 이동한다.

· A (교번): 0과 1이 균등 → 기준선 안정, 오류 낮음
· B (연속 1): 기준선이 지속적으로 음 방향으로 표류 → 가장 위험
· C, D: 혼합/짧은 반복 → 중간 수준의 위험

계산형 [응용] 역방향 ⭐⭐⭐ Slide 17
어떤 PCM 시스템의 비트율(Bit Rate)이 176,400 bps이고, 표본화율이 44,100 samples/sec일 때, 표본당 비트 수(nb)는 얼마인가? 그리고 이 시스템이 표현할 수 있는 양자화 구간 수(L)는?
✅ 정답
nb = 4 bits, L = 16구간
풀이 과정
PCM 공식: Bit Rate = fs × nb 를 nb에 대해 역산
nb = Bit Rate / fs
nb = 176,400 / 44,100 = 4 bits
양자화 구간 수:
L = 2nb = 24 = 16 구간
해석: 이 시스템은 44,100 Hz 표본화율(나이퀴스트 기준 최대 22,050 Hz 재생 가능)에 4비트(16단계) 양자화를 사용한다. 실제 CD 음질(16비트)에 비해 양자화 단계가 매우 적어 음질이 낮다.

오버헤드 비율:

직렬 전송이 유리한 상황:

② 비용 절감: 직렬은 병렬 대비 1/n 비용으로 동일 데이터 전송 가능
③ 원거리 네트워크: 인터넷, 모뎀 등 장거리 환경에서 실질적으로 직렬만 사용 가능

다음 조건을 모두 만족하는 PCM 시스템을 설계하시오.

• 최대 신호 주파수: 8,000 Hz
• 양자화 구간 수: 256 단계
• 신호 요소 1개가 데이터 요소 1개를 표현 (r = 1)
① PCM 비트율(Bit Rate)을 구하시오.
② 신호 전송률(S, baud)을 구하시오.
③ 비트율을 절반으로 줄이려면 양자화 구간 수를 어떻게 바꿔야 하는지, 그 부작용은 무엇인지 설명하시오.
✅ 정답 및 풀이
① PCM 비트율
fs = 2 × 8,000 = 16,000 samples/sec
nb = log₂256 = 8 bits
Bit Rate = 16,000 × 8 = 128,000 bps = 128 Kbps
② 신호 전송률
r = 1 이므로 S = N / r = 128,000 / 1 = 128,000 baud
③ 비트율 절반으로 줄이기
목표 Bit Rate = 64,000 bps
nb = 64,000 / 16,000 = 4 bits
→ L = 24 = 16 구간으로 줄여야 함 (256 → 16)
부작용: 양자화 구간 수가 256→16으로 크게 줄어들면 각 구간의 높이(Δ)가 16배 커진다. 이로 인해 양자화 잡음(Quantization Noise)이 크게 증가하여 원래 아날로그 신호와의 차이(왜곡)가 심해지고 음질/신호 품질이 현저히 저하된다.
