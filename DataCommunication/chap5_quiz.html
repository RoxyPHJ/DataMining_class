객관식 QPSK 파형 ⭐⭐ 보통 Slide 14
QPSK에서 I채널(동위상)과 Q채널(구상)의 파형이 아래와 같다.
두 신호를 합성한 최종 QPSK 신호의 진폭은 얼마인가?

I(t) = A·cos(2πf꜀t),  Q(t) = A·sin(2πf꜀t)
A A (단일 진폭, 위상만 변함)
B √2 · A (두 채널 합성으로 진폭 증가)
C 2A (단순 합산)
D 0 (위상차 90°로 상쇄)
정답
B — √2 · A
QPSK는 I채널(A·cos)과 Q채널(A·sin)을 합성한다.
합성 신호: S(t) = A·cos + A·sin = √2·A · cos(2πf꜀t − 45°)
진폭의 크기: √(A² + A²) = √2·A
→ 위상은 45° 이동하지만 진폭은 √2배가 된다. 두 채널이 직교(90° 차이)하므로 에너지가 더해진다.

객관식 QPSK 파형 [응용] ⭐⭐⭐ 어려움 Slide 14-15
아래는 QPSK 전송 시 I채널과 Q채널의 파형이다.
1구간 (비트 11)과 2구간 (비트 00)에서 합성된 최종 신호의 위상 변화로 옳은 것은?

• 1구간: I = +A·cos, Q = +A·sin (둘 다 양수)
• 2구간: I = −A·cos, Q = −A·sin (둘 다 음수)
A 1구간: 45°, 2구간: 45° (위상 변화 없음)
B 1구간: 45°, 2구간: 90°
C 1구간: 45°, 2구간: 225° (위상 180° 점프)
D 1구간: 135°, 2구간: 315°
정답
C — 1구간: 45°, 2구간: 225°
합성 위상 = arctan(Q/I)
• 1구간: I=+A, Q=+A → arctan(+/+) = 45°
• 2구간: I=−A, Q=−A → arctan(−/−) = 180°+45° = 225°
→ 비트 11(45°)에서 비트 00(225°)으로 이동 시 위상이 180° 점프한다.
이것이 QPSK의 핵심: 2비트 변화마다 위상이 크게 바뀔 수 있다.
성운도에서 보면 (45°)→(225°) 이동은 대각선 반대편으로 이동하는 것과 같다.

계산형 QAM 계산 [응용] ⭐⭐ 보통 Slide 17
256-QAM을 사용하여 비트율 N = 16,000 bps로 데이터를 전송하려 한다.
필요한 신호율 S (Baud)를 구하라.
정답
S = 2,000 Baud
r = log₂ 256 = 8
S = N / r = 16,000 / 8 = 2,000 Baud
[응용 개념] 역방향 문제: 비트율과 변조 방식이 주어졌을 때 필요한 신호율을 구한다.
256-QAM은 신호 하나에 8비트를 담으므로 신호율은 비트율의 1/8이 된다.

서술형 ⭐⭐⭐ 어려움 Slide 8, 10
ASK가 FSK보다 잡음에 취약한 이유를 두 변조 방식의 특성을 비교하여 서술하라.
(2~3문장)
모범 답안
ASK는 진폭의 변화로 0과 1을 구분하므로, 전송 중 잡음이 신호의 세기(진폭)를 변화시키면 수신측에서 0인지 1인지 판별하기 어렵다.

반면 FSK는 주파수의 변화로 데이터를 표현하므로, 잡음이 진폭을 바꿔도 주파수 자체는 크게 영향받지 않아 수신측이 더 정확하게 판별할 수 있다.

따라서 FSK는 ASK에 비해 SNR(Signal-to-Noise Ratio)이 낮은 환경에서도 더 안정적으로 동작한다.
