Chapter 03 — 물리층 개요 (1)

데이터와 신호

아날로그/디지털 데이터와 신호의 특성, 주기 아날로그 신호, 복합신호, 대역폭, 디지털 신호 전송 방식을 학습합니다.

강의자료 chap3-1 섹션 10개 슬라이드 1–18
01

데이터와 신호

데이터(Data) 유형
  • 아날로그 데이터 — 연속적인 값을 갖는 데이터 (예: 음성, 온도, 시간)
  • 디지털 데이터 — 0과 1로 저장된 이산적 데이터
신호(Signal) 유형
  • 아날로그 신호 (Analog Signal) — 연속적인 파형으로 표현. 값이 매끄럽게 변화.
  • 디지털 신호 (Digital Signal) — 이산적이며 1, 0과 같이 제한된 수의 정의된 값만을 가짐.
⚠️ 데이터와 신호는 독립적으로 조합 가능 — 아날로그 데이터를 디지털 신호로 전송(코덱), 디지털 데이터를 아날로그 신호로 전송(모뎀)할 수 있습니다.
데이터–신호 변환 조합 (슬라이드 3)
02

주기 아날로그 신호

정현파 (Sine Wave) — 슬라이드 4

싸인파(정현파)는 아날로그 주기 신호의 가장 기본적인 형태입니다.

s(t) = A · sin(2πft + φ)
// s: 순간 진폭, A: 최대 진폭
// f: 주파수(Hz), φ: 위상(radian)
  • 진폭 (Amplitude, A) — 신호의 최대 크기 (V 또는 W)
  • 주파수 (Frequency, f) — 1초당 사이클 수 (Hz)
  • 위상 (Phase, φ) — 기준점 대비 파형의 이동량 (°또는 rad)
  • 주기 (Period, T) — 한 사이클의 시간, T = 1/f
위상(Phase) 비교
  • 위상은 시간 기원에 대한 파형의 상대적 위치를 나타냄
  • 단위: 도(°) 또는 라디안(rad)
  • φ = 0° 이면 기준, φ = 90° 이면 1/4 주기 앞섬
  • φ = 180° 이면 역위상 (180° 차이 = 완전히 반대)
03

파장 (Wavelength)

파장 개념 — 슬라이드 5–6

파장(λ)이란 단순 신호가 한 주기 동안 진행할 수 있는 거리입니다.

파장은 전파 속도(propagation speed)와 주기를 이용해 계산합니다.

λ = c / f
// λ: 파장(m), c: 전파속도(m/s)
// f: 주파수(Hz)
예제: 빨간색 빛 (f = 4 × 10¹⁴ Hz)
c = 3 × 10⁸ m/s (진공에서 빛의 속도)
λ = 3×10⁸ / 4×10¹⁴ = 7.5 × 10⁻⁷ m = 750 nm
04

시간 영역과 주파수 영역

두 가지 표현 방식 — 슬라이드 7
  • 시간 영역 (Time Domain) — 시간 축 기준, 신호의 진폭이 시간에 따라 변하는 형태. 오실로스코프로 관측.
  • 주파수 영역 (Frequency Domain) — 주파수 축 기준, 각 주파수 성분의 진폭을 표현. 스펙트럼 분석기로 관측.
단순 신호는 주파수 영역에서 단 하나의 막대(spike)로 표현됩니다.
05

복합 신호

복합신호 종류 — 슬라이드 8–10

실제 데이터 통신에 사용되는 신호는 모두 복합신호(Composite Signal)입니다.

주기적 복합신호
반복되는 패턴. 주파수 영역에서 이산적인 스파이크들로 표현.
비주기적 복합신호
반복 패턴 없음. 주파수 영역에서 연속적인 스펙트럼으로 표현.
주기 복합신호의 분해 — 슬라이드 8–9

복합신호는 여러 단순 정현파의 합으로 분해할 수 있습니다 (푸리에 급수).

x(t) = A₁sin(2πf₁t) + A₂sin(2πf₂t) + ...
  • 각 성분은 서로 다른 주파수, 진폭, 위상을 가짐
  • 주파수 영역에서 각 성분이 개별 스파이크로 표현
06

대역폭 (Bandwidth)

대역폭 정의 — 슬라이드 11

복합신호의 대역폭(Bandwidth)이란 신호에 포함된 최고 주파수와 최저 주파수의 차입니다.

B = f_max − f_min
// B: 대역폭(Hz)
// f_max: 최고 주파수, f_min: 최저 주파수
  • 주기 복합신호 — 이산 주파수들의 범위
  • 비주기 복합신호 — 연속 스펙트럼의 범위
07

디지털 신호

디지털 신호의 특성 — 슬라이드 12
  • 디지털 신호는 무한대의 주파수를 갖는 복합 신호로 볼 수 있음
  • 이상적인 디지털 신호의 대역폭은 무한대 (∞)
  • 실제 전송 시 유한한 대역폭으로 근사
⚠️ 대역폭이 무한하므로 실제 채널에서는 고주파 성분이 손실 → 신호 왜곡 발생
08

기저대역 전송

Baseband 전송 — 슬라이드 13–16

기저대역(Baseband) 전송이란 디지털 신호를 아날로그 신호로 변환하지 않고 있는 그대로 채널을 통해 전송하는 방식입니다.

  • 전용 매체(dedicated medium) 사용 필요
  • 저대역 통과 채널(Low-pass channel) 사용 — 주파수 0부터 시작하는 대역폭 보유
  • 동축케이블, 광섬유에서 사용
  • LAN(근거리 통신망)에서 주로 사용
  • 약간의 오차는 추론으로 보완 가능
특징: f₁ = 0 (주파수가 0부터 시작)
전용 매체 baseband 전송에 사용
09

광대역 전송 (변조)

Wideband / Bandpass 전송 — 슬라이드 17–18

광대역(Wideband) 전송은 디지털 신호를 전송하기 위해 아날로그 신호로 변환(변조)하는 방식입니다.

  • 변조(Modulation)를 통해 밴드패스(Bandpass) 채널을 사용하여 전송
  • 밴드패스 채널의 대역폭: f_min ≠ 0
  • 주로 공중망(PSTN), 무선통신에서 사용
B = f_max − f_min
// 밴드패스 채널 대역폭
변조 방식: 디지털 데이터 → 아날로그 반송파(carrier)에 실어 전송
예: AM, FM, 위상변조(PM) 등
기저대역 vs 광대역 비교
구분 기저대역 (Baseband) 광대역 (Wideband)
신호 변환 없음 (디지털 그대로) 변조 필요 (디지털→아날로그)
채널 유형 저대역 통과 (Low-pass) 밴드패스 (Bandpass)
주파수 시작점 0 Hz (f₁ = 0) 0 이상 (f_min ≠ 0)
전용 매체 필요 불필요 (공유 가능)
주요 사용처 LAN, 동축케이블, 광섬유 전화망(PSTN), 무선통신
장치 예 이더넷 NIC 모뎀
10

핵심 요약

01 — 데이터와 신호
아날로그 & 디지털
데이터(아날로그/디지털)와 신호(아날로그/디지털)는 독립적으로 조합. 모뎀·코덱으로 변환.
02 — 정현파
진폭, 주파수, 위상
s(t) = A·sin(2πft + φ). T = 1/f. 아날로그 신호의 기본 구성 요소.
03 — 파장
λ = c / f
한 주기 동안 진행 거리. 빛 속도 c = 3×10⁸ m/s. 적색광 λ ≈ 750 nm.
04 — 시·주파수 영역
이중 표현
단순 신호 → 주파수 영역에서 1개 스파이크. 복합 신호 → 여러 스파이크.
05 — 복합신호
실제 신호 구성
주기/비주기 복합신호로 구분. 단순 정현파의 합으로 표현 (푸리에 급수).
06 — 대역폭
B = f_max − f_min
신호 최고·최저 주파수의 차. 주기는 이산, 비주기는 연속 스펙트럼.
07 — 디지털 신호
무한 대역폭
이론적 대역폭 ∞. 실제 전송시 유한 대역폭으로 근사 → 고주파 손실·왜곡.
08 — 기저대역
변환 없이 전송
디지털 신호 그대로, Low-pass 채널, LAN/광섬유 사용. f₁ = 0.
09 — 광대역
변조 후 전송
변조로 Bandpass 채널 활용. 전화망·무선통신에 사용. 모뎀 필요.
