객관식
Q1 – Q12 · 12문제
Q01 ⭐ p.4
"data"라는 단어의 어원으로 올바른 것은?
정답
B — 라틴어 "datum"의 복수형
data는 datum(주어진 것)의 복수형으로, 컴퓨터 분야에서 transmissible and storable computer information을 의미하는 말로 처음 사용되었습니다.
Q02 ⭐ p.4
"data processing"이라는 표현이 처음 사용된 연도는?
정답
C — 1954년
슬라이드 p.4에 명시된 내용으로, "the expression data processing was first used in 1954"입니다.
Q03 ⭐ p.5
데이터셋에서 각 행(row)을 나타내며, 단일 개체를 설명하는 요소를 무엇이라 하는가?
정답
C — Instance
Instance는 데이터셋의 각 행을 나타내며 object, record, point, vector, sample 등의 별칭도 가집니다. Attribute는 각 열(column)을 나타냅니다.
Q04 ⭐⭐ p.6
다음 중 Ordinal 데이터의 예로 가장 적절한 것은?
정답
C — 통증 강도 (경미, 보통, 심각)
Ordinal은 순서(hierarchy)가 존재하는 범주형 데이터입니다. 통증 강도·만족도·등급 등이 해당됩니다. 혈액형·눈 색깔은 Nominal, 자녀 수는 Discrete(수치형)입니다.
Q05 ⭐⭐ p.7
ImageNet에 대한 설명으로 옳지 않은 것은?
정답
D — 비디오 이해에 사용된다 (틀림)
ImageNet은 이미지 분류(visual object recognition) 벤치마크 데이터셋입니다. 비디오 이해(action recognition)에 사용되는 데이터셋은 Kinetics입니다.
Q06 ⭐⭐ p.8
MS COCO 데이터셋이 지원하는 태스크로 옳지 않은 것은?
정답
C — 3D Point Cloud Labeling
MS COCO는 Object Detection, Segmentation, Keypoint Detection, Image Captioning을 지원합니다. 3D Point Cloud Labeling은 자율주행 어노테이션(LiDAR) 기법으로 COCO의 지원 태스크가 아닙니다.
Q07 ⭐⭐ p.10
Kinetics 데이터셋에 대한 설명으로 옳은 것은?
정답
B — 600개 행동 클래스에 각각 최소 600개 클립
Kinetics는 인간 행동 인식(action recognition) 및 비디오 이해를 위한 데이터셋으로, 총 60만 개 비디오 클립, 600개 클래스, 클래스당 최소 600 클립으로 구성됩니다.
Q08 ⭐⭐ p.17
CVAT(Computer Vision Annotation Tool)의 특징으로 옳지 않은 것은?
정답
C — Bounding Box만 지원한다 (틀림)
CVAT는 Object Detection, Semantic Segmentation, Instance Segmentation, Tracking 등 다양한 태스크를 지원합니다. Bounding Box만 지원한다는 설명은 틀렸습니다.
Q09 ⭐⭐⭐ p.23
EDA의 4원칙 중 "Resistance"에 해당하는 실천 방법으로 가장 적절한 것은?
정답
B — median과 IQR 사용
Resistance는 분석이 이상치·오류에 강건해야 한다는 원칙입니다. 평균·표준편차는 이상치에 민감하므로 중앙값(median)과 사분위범위(IQR)를 사용합니다. A는 Revelation, C는 Re-expression, D는 Residuals에 해당합니다.
Q10 ⭐⭐⭐ p.26
다음 데이터 전처리 기법과 그 목적이 바르게 연결된 것은?
정답
C — One-hot Encoding → 범주형 변수를 수치형으로 변환
One-hot Encoding은 범주형 변수를 0/1의 이진 벡터로 변환하는 Data Transformation 기법입니다. 정규화는 스케일 조정, 결측값 처리는 Data Cleansing, PCA는 Data Reduction에 해당합니다.
Q11 ⭐⭐⭐ p.8–9
다음 데이터셋 중 이미지 수가 가장 많은 것은?
정답
D — ImageNet (약 1,400만 장)
ImageNet ≈ 1,400만 장 > MS COCO ≈ 330,000장 > ADE 20k ≈ 20,210장 > PASCAL VOC ≈ 10,000장 순입니다.
Q12 ⭐⭐⭐ p.26–27
Data Preprocessing의 4가지 핵심 작업을 올바르게 나열한 것은?
정답
A — Cleansing / Integration / Transformation / Reduction
Data Preprocessing의 4가지 핵심 작업은 Data Cleansing(결측·노이즈 처리), Data Integration(소스 통합), Data Transformation(정규화·인코딩), Data Reduction(차원 축소)입니다. B~D는 세부 기법이거나 다른 단계입니다.

Q13 ⭐ p.4
big data 문맥에서 "data"는 일반적으로 "information"의 동의어로 사용된다.

Q14 ⭐ p.5
데이터셋에서 Attribute는 각 행(row)을 나타내며, 단일 개체의 특성을 설명한다.
정답
X (거짓)
Attribute는 각 열(column)을 나타냅니다. 각 행(row)을 나타내는 것은 Instance입니다.
Q15 ⭐ p.6
Continuous 데이터는 유한한(finite) 옵션을 가지는 수치형 데이터이다.
정답
X (거짓)
Continuous 데이터는 무한한(infinite) 옵션을 갖습니다. 유한한(finite) 옵션을 갖는 것은 Discrete 데이터입니다. (예: 나이·체중·혈압 vs 신발 사이즈·자녀 수)
Q16 ⭐⭐ p.8
MS COCO는 현재 딥러닝에서 가장 영향력 있고 널리 사용되는 데이터셋 중 하나로, 80개의 카테고리와 150만 개의 객체 인스턴스를 포함한다.
정답
O (참)
슬라이드 p.8에 MS COCO는 "one of the most influential and widely used datasets", 80 categories, 1.5M object instances라고 명시되어 있습니다.
Q17 ⭐⭐ p.18
LabelMe는 labelme2coco 도구를 통해 어노테이션 데이터를 COCO 포맷으로 변환·내보낼 수 있다.
정답
O (참)
슬라이드 p.18에 "labelme2coco helps to export data annotation as COCO format"이라고 명시되어 있습니다.
Q18 ⭐⭐ p.23
EDA는 일반적으로 공식적인 모델링(formal modeling) 이후에 적용되는 단계이다.
정답
X (거짓)
EDA는 공식적인 모델링 이전(before formal modeling)에 적용됩니다. 데이터를 먼저 탐색·이해한 후 모델링에 들어갑니다.
Q19 ⭐⭐ p.23
EDA의 Re-expression 원칙은 왜도가 큰 데이터에 로그 변환을 적용하여 패턴을 더 명확하게 드러내는 것을 포함한다.
정답
O (참)
Re-expression은 데이터를 변환·재구성하여 패턴을 더 명확히 드러내는 것입니다. 로그 변환으로 왜곡된 데이터를 정규화하는 것이 대표적인 예입니다.
Q20 ⭐⭐⭐ p.28
실세계 데이터의 특성 중 "Noisy"란 데이터에 오류나 이상치(outlier)가 포함된 상태를 의미한다.
정답
O (참)
슬라이드 p.28에서 Noisy는 "data contains errors or outliers"로 정의됩니다. Incomplete는 결측, Inconsistent는 코드·이름 불일치를 의미합니다.
Q21 ⭐⭐⭐ p.29
설문 결과에 따르면 데이터 과학자들이 가장 즐겁지 않다고 꼽은 작업 1위와 실제 가장 많은 시간을 소비하는 작업 1위가 모두 데이터 정제(Cleaning)이다.
정답
O (참)
슬라이드 p.29의 두 차트 모두에서 "Cleaning and organizing data"가 1위(각각 57%, 60%)를 차지합니다. 가장 즐겁지 않으면서도 가장 많은 시간이 소요되는 아이러니한 현실입니다.
Q22 ⭐⭐⭐ p.9
ADE 20k 데이터셋은 주로 객체 탐지(Object Detection)를 위한 데이터셋으로, 20,210장의 이미지와 20개의 카테고리를 포함한다.
정답
X (거짓)
ADE 20k는 주로 의미론적 분할(Semantic Segmentation)을 위한 데이터셋입니다. 이미지 수 20,210장은 맞지만, 카테고리는 20개가 아니라 150개 이상(150+ categories)입니다. 20개 카테고리는 PASCAL VOC의 수치입니다.

Q23 – Q30 · 8문제
Q23 ⭐ p.4
DIKW 피라미드에서 가장 하위 단계(가장 기초적인 레벨)는 무엇인가?
정답
Data
DIKW 피라미드는 Data → Information → Knowledge → Wisdom 순으로 계층을 이룹니다. Data가 가장 하위 단계이며, 원시적인 이산값들로 구성됩니다.
Q24 ⭐ p.6
혈액형(A, B, O, AB)처럼 순서(계층)가 없는 범주형 데이터를 무엇이라 하는가?
정답
Nominal (명목형)
Nominal 데이터는 순서(hierarchy)가 없는 범주형 데이터입니다. 순서가 있으면 Ordinal입니다. 예: 혈액형, 눈 색깔, 강아지 품종 등.
Q25 ⭐ p.7
ImageNet 기반으로 개최되는 대규모 이미지 분류 챌린지의 약어(4글자)는?
정답
ILSVRC
ImageNet Large Scale Visual Recognition Challenge의 약어입니다. 딥러닝 모델의 이미지 분류 성능을 벤치마킹하는 대회로, AlexNet(2012) 등 주요 모델들이 이 대회를 통해 발표되었습니다.
Q26 ⭐⭐ p.17
웹 기반 컴퓨터 비전 어노테이션 도구로, COCO·PASCAL VOC·YOLO 포맷 내보내기를 지원하는 툴의 이름은? (영문 약어)
정답
CVAT
Computer Vision Annotation Tool의 약어입니다. 웹 기반 인터페이스를 제공하며 object detection, semantic segmentation, instance segmentation, tracking을 지원합니다.
Q27 ⭐⭐ p.23
EDA(탐색적 데이터 분석)의 개념을 정립한 통계학자의 이름은? (성(last name)만 작성)
정답
Tukey
John Wilder Tukey (1915–2000)가 EDA를 정립했습니다. 그는 Box Plot을 발명하고 데이터를 먼저 탐색한 후 모델링에 들어가는 패러다임을 확립했습니다.
Q28 ⭐⭐ p.23
EDA의 4원칙 중, 관측값과 예측값의 차이를 분석하여 모델의 적합도를 평가하는 원칙은?
정답
Residuals (잔차)
Residuals는 관측값(observed values)과 예측값(predicted values)의 차이를 분석하여 모델이 데이터에 얼마나 잘 맞는지(goodness of fit) 평가합니다.
Q29 ⭐⭐⭐ p.26
Data Preprocessing에서 여러 소스의 데이터를 하나의 통합된 데이터셋으로 결합하는 작업을 무엇이라 하는가?
정답
Data Integration
Data Integration은 여러 소스(databases, flat files 등)의 데이터를 하나의 통합된 데이터셋으로 결합하는 작업입니다. 이는 전처리의 4가지 핵심 작업 중 하나입니다.
Q30 ⭐⭐⭐ p.28
EDA의 Resistance 원칙에서 평균(mean) 대신 사용하는 대표값과, 표준편차(std) 대신 사용하는 산포 지표를 각각 쓰시오. (형식: A, B)
정답
median (중앙값), IQR (사분위범위 / Interquartile Range)
Resistance 원칙은 이상치·오류에 강건한 분석을 요구합니다. 평균(mean)과 표준편차(std)는 이상치에 민감하므로, 이상치에 강건한 중앙값(median)과 IQR(Interquartile Range = Q3 - Q1)을 사용합니다.
