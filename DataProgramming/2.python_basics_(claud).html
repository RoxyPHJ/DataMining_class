Variables

변수란?

데이터 값을 저장하는 컨테이너. 값을 처음 할당하는 순간 생성되며, 특정 타입으로 선언할 필요 없음.

Naming Convention

  • 문자 또는 _로 시작
  • 숫자로 시작 불가
  • 영숫자와 _만 포함 가능
  • 대소문자 구분: age, Age, AGE는 모두 다른 변수

✅ Legalmyvar, my_var, _my_var, myVar, MYVAR, myvar2
❌ Illegal2myvar, my-var, my var (공백/하이픈/숫자 시작 불가)
x = 5 y = "John" print(x) print(y)

파이썬 내장 데이터 타입

Casting

변수 타입 명시적 지정

Python은 동적 타입 언어이지만, 캐스팅 함수로 타입을 명시적으로 변환할 수 있다.

int() — 정수형 변환
x = int(1) # x = 1 y = int(2.8) # y = 2 (소수점 버림) z = int("3") # z = 3 (문자열→정수)
float() — 실수형 변환
x = float(1) # x = 1.0 y = float(2.8) # y = 2.8 z = float("3") # z = 3.0 w = float("4.2") # w = 4.2
str() — 문자열 변환
x = str("s1") # x = 's1' y = str(2) # y = '2' z = str(3.0) # z = '3.0'

String

기본 개념

문자열은 단일/이중 따옴표로 감싸며 서로 동일: 'hello' == "hello"

문자열은 유니코드 문자 바이트 배열이므로 인덱스로 접근 가능.

기본 조작
a = "Hello, World!" print(a[1]) # 인덱스 접근 print(len(a)) # 문자열 길이 # 포함 여부 확인 txt = "The best things in life are free!" if "free" in txt: print("Yes, 'free' is present.")
슬라이싱
b = "Hello, World!" print(b[2:5]) # "llo" print(b[:5]) # "Hello" print(b[2:]) # "llo, World!" print(b[-5:-2]) # 음수 인덱스
문자열 메서드
a = "Hello, World!" print(a.upper()) # 대문자 print(a.lower()) # 소문자 print(a.strip()) # 앞뒤 공백 제거 print(a.replace("H", "J")) # 치환 print(a.split(",")) # 분할 → ['Hello', ' World!']
String Format — format() 메서드
❌ 오류 발생Python은 문자열과 숫자를 직접 합칠 수 없다: "name" + 36 → TypeError
age = 36 txt = "My name is John, and I am {}" print(txt.format(age)) # 여러 플레이스홀더 quantity = 3 itemno = 567 price = 49.95 myorder = "I want {} pieces of item {} for {} dollars." print(myorder.format(quantity, itemno, price))

List

특징Ordered(순서 있음) · Changeable(변경 가능) · Duplicates Allowed(중복 허용) — 대괄호 [] 사용
생성 & 접근
thislist = ["apple", "banana", "cherry"] print(thislist) print(thislist[1]) # "banana" # list() 생성자 thislist = list(("apple", "banana", "cherry"))
항목 변경
thislist[1] = "blackcurrant" print(thislist)
항목 추가
# append(): 끝에 추가 thislist.append("orange") # insert(): 특정 인덱스에 삽입 thislist.insert(2, "watermelon") # extend(): 다른 리스트 요소 전체 추가 tropical = ["mango", "pineapple", "papaya"] thislist.extend(tropical)
항목 제거
thislist.remove("banana") # 특정 값 제거 thislist.pop(1) # 특정 인덱스 제거 thislist.pop() # 마지막 항목 제거 del thislist[0] # del로 특정 인덱스 제거 del thislist # 리스트 자체 삭제 thislist.clear() # 리스트 비우기

Tuple

특징Ordered(순서 있음) · Unchangeable(변경 불가, Immutable) · Duplicates Allowed(중복 허용) — 소괄호 () 사용
생성 & 접근
thistuple = ("apple", "banana", "cherry") print(thistuple[1]) # tuple() 생성자 thistuple = tuple(("apple", "banana", "cherry"))
값 변경 방법 (우회)
⚠️ Immutable직접 변경 불가. List로 변환 후 수정.
x = ("apple", "banana", "cherry") y = list(x) y[1] = "kiwi" x = tuple(y) print(x)
항목 추가 (우회) & Tuple + Tuple
# 리스트 변환 후 추가 thistuple = ("apple", "banana", "cherry") y = list(thistuple) y.append("orange") thistuple = tuple(y) # 튜플끼리 합치기 y = ("orange",) thistuple += y

Set

특징Unordered(순서 없음) · Unchangeable(변경 불가) · No Duplicates(중복 불가) — 중괄호 {} 사용
생성 & 순회
thisset = {"apple", "banana", "cherry"} print(thisset) # 인덱스로 접근 불가 → for 루프로 순회 for x in thisset: print(x)
항목 추가
# 단일 항목 추가 thisset.add("orange") # 다른 set 병합 tropical = {"pineapple", "mango", "papaya"} thisset.update(tropical) print(thisset)
항목 제거
thisset.remove("banana") # 없으면 오류 발생 thisset.discard("banana") # 없어도 오류 없음 x = thisset.pop() # 랜덤 항목 제거 후 반환 thisset.clear() # set 비우기 del thisset # set 완전 삭제

Dictionary

특징Key:Value 쌍으로 데이터 저장 · Ordered/Unordered(Python 버전에 따라 다름) · Changeable · No Duplicates — 중괄호 {} 사용
생성
thisdict = { "brand": "Ford", "model": "Mustang", "year": 1964 } print(thisdict) # dict() 생성자 thisdict = dict(name="John", age=36, country="Norway")
항목 접근
x = thisdict["model"] # 직접 접근 x = thisdict.get("model") # get() x = thisdict.keys() # 모든 키 x = thisdict.values() # 모든 값 x = thisdict.items() # 모든 key-value 쌍
항목 변경 · 추가 · 제거
# 값 변경 thisdict["year"] = 2018 thisdict.update({"year": 2020}) # 새 항목 추가 thisdict["color"] = "red" # 항목 제거 thisdict.pop("model") # 특정 키 제거 del thisdict["model"] # del로 제거 thisdict.clear() # 비우기

While Loops

조건이 참인 동안 반복 실행
# 기본 while i = 1 while i < 6: print(i) i += 1
break — 루프 강제 종료
i = 1 while i < 6: print(i) if i == 3: break i += 1
continue & else
# continue: 현재 반복 건너뜀 i = 0 while i < 6: i += 1 if i == 3: continue print(i) # else: 조건이 거짓이 될 때 실행 i = 1 while i < 6: print(i) i += 1 else: print("i is no longer less than 6")

Functions

함수 정의 & 인수
# 기본 함수 정의 def my_function(): print("Hello from a function") my_function() # 인수 전달 def my_function(fname, lname): print(fname + " " + lname) my_function("Emil", "Refsnes")
기본값 & return
# 기본 파라미터 값 def my_function(country = "Norway"): print("I am from " + country) my_function() # → "I am from Norway" my_function("Sweden") # → "I am from Sweden" # return 값 반환 def my_function(x): return 5 * x print(my_function(3)) # → 15
*args — 가변 위치 인수 (tuple)
def my_function(*kids): print("The youngest child is " + kids[2]) my_function("Emil", "Tobias", "Linus") # args는 tuple → kids[2] = "Linus"
**kwargs — 가변 키워드 인수 (dict)
def my_function(**kid): print("His last name is " + kid["lname"]) my_function(fname="Tobias", lname="Refsnes") # kwargs는 dict → kid["lname"]
재귀 (Recursion) — 주의 필요!
⚠️ 주의재귀는 무한 루프, 메모리/성능 문제를 일으킬 수 있다. 올바르게 작성했을 때 매우 효율적이다.
def tri_recursion(k): if(k > 0): result = k + tri_recursion(k - 1) print(result) else: result = 0 return result tri_recursion(6) # 1, 3, 6, 10, 15, 21 출력

Inheritance

상속이란?
  • OOP의 핵심 기능 — 코드 재사용성 향상
  • 부모 클래스(Parent/Base/Superclass): 속성과 메서드를 제공하는 클래스
  • 자식 클래스(Child/Derived/Subclass): 부모로부터 상속받는 클래스
super() — 부모 __init__ 호출
핵심자식 클래스에서 __init__을 재정의할 때 super().__init__()으로 부모 초기화를 유지해야 한다.
상속 예제
# 부모 클래스 (Parent class / Base class / Superclass) class Animal: def __init__(self, name): self.name = name def speak(self): print(f"{self.name} makes a sound") # 자식 클래스 (Child class / Derived class / Subclass) class Dog(Animal): def __init__(self, name, breed): super().__init__(name) # 부모 생성자 호출 self.breed = breed def speak(self): # 메서드 오버라이드 print(f"{self.name} the {self.breed} barks!")
