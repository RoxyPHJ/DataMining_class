01
선형대수 — 벡터 · 평면 · 직교
원본 참조: 문제 1 [①②③]
[응용] 1-A ⭐⭐ 보통 수치 변형
평면 P를 span하는 두 벡터가 아래와 같이 주어진다.
u₁ = (1, 0, 1)ᵀ, u₂ = (0, 1, 1)ᵀ

두 벡터 a와 b가 다음과 같다.
a = (1, 1, −2)ᵀ, b = (1, 1, 2)ᵀ

(1) ‖a‖와 ‖b‖를 구하라.
(2) 내적 a·b와 cos θ를 구하라.
(3) a·u₁, a·u₂를 계산하고 a와 평면 P의 관계를 설명하라.
(4) b·u₁, b·u₂를 계산하고 b가 평면 P에 속하는지 확인하라.
(5) a와 b의 기하학적 관계를 한 문장으로 서술하라.
▼
▶ 정답 및 풀이
(1) 노름 계산
‖a‖ = √(1² + 1² + (−2)²) = √6 ‖b‖ = √(1² + 1² + 2²) = √6
(2) 내적 및 cos θ
a · b = (1)(1) + (1)(1) + (−2)(2) = 1 + 1 − 4 = −2 cos θ = −2 / (√6 · √6) = −2/6 = −1/3 ≈ −0.333
(3) a와 평면 P의 관계
a · u₁ = 1·1 + 1·0 + (−2)·1 = 1 + 0 − 2 = −1 ≠ 0 a · u₂ = 1·0 + 1·1 + (−2)·1 = 0 + 1 − 2 = −1 ≠ 0 → 둘 다 0이 아니므로 a는 평면 P에 직교하지 않음. 평면 P 밖에 있으며 P의 법선벡터도 아님.
(4) b와 평면 P의 관계
b · u₁ = 1·1 + 1·0 + 2·1 = 1 + 0 + 2 = 3 ≠ 0 b · u₂ = 1·0 + 1·1 + 2·1 = 0 + 1 + 2 = 3 ≠ 0 → b는 평면 P의 원소가 아님. u₁, u₂에 대해 모두 내적이 0이 아니므로 P 위에 없음. 단, b = 1·u₁ + 1·u₂ = (1,0,1)+(0,1,1) = (1,1,2) ✓이므로 b = u₁ + u₂로 b ∈ P임을 직접 확인 가능. ※ 내적≠0이 "P 밖"을 의미하지 않음 — P 안 벡터가 u₁,u₂와 직교해야 할 이유는 없음.
(5) 종합
a는 평면 P 밖에 있고, b = u₁+u₂로 평면 P 위에 있으며, cos θ = −1/3으로 두 벡터는 둔각을 이룬다.
📌 원본 대비 변형 포인트
원본의 u₁=(1,1,0), u₂=(1,0,1), a=(1,−1,−1), b=(2,1,1)에서
→ u₁=(1,0,1), u₂=(0,1,1)으로 평면 방향 변경
→ a=(1,1,−2)로 변경 (직교 여부 구조 유지하되 내적 결과 ≠ 0이 되도록)
→ b=u₁+u₂=(1,1,2)로 정확히 P 위에 놓이도록 설계
[응용] 1-B ⭐⭐⭐ 어려움 역방향 문제
평면 P가 두 벡터 u₁ = (1, 2, 0)ᵀ, u₂ = (0, 1, 1)ᵀ에 의해 span된다.
벡터 v = (a, 1, −1)ᵀ가 평면 P에 포함되기 위한 a의 값을 구하라.

(1) v ∈ P이면 v = αu₁ + βu₂ 형태로 쓸 수 있음을 이용하여 연립방정식을 세워라.
(2) 연립방정식을 풀어 α, β, a를 구하라.
(3) 구한 a로 v를 완성하고, v가 평면 P에 직교하는지 판별하라.
▼
▶ 정답 및 풀이
(1) 연립방정식 설정
v = αu₁ + βu₂: α·1 + β·0 = a → α = a … ① α·2 + β·1 = 1 → 2a + β = 1 … ② α·0 + β·1 = −1 → β = −1 … ③
(2) 풀이
③에서 β = −1 ②에 대입: 2a + (−1) = 1 → 2a = 2 → a = 1 ①에서 α = 1 검증: 1·(1,2,0) + (−1)·(0,1,1) = (1, 2−1, 0−1) = (1, 1, −1) = v ✓
(3) 직교 판별
v = (1, 1, −1)ᵀ v · u₁ = 1·1 + 1·2 + (−1)·0 = 3 ≠ 0 v · u₂ = 1·0 + 1·1 + (−1)·1 = 0 → v·u₁ ≠ 0이므로 v는 평면 P에 직교하지 않는다. (v는 P 위에 있으므로 직교할 수 없음)
📌 원본 대비 변형 포인트
원본: 벡터가 주어지고 → 내적/관계를 출력
응용: 역방향 — 평면 포함 조건과 벡터 일부 성분이 미지수로 주어지고 → 해당 값을 역산
풀이 구조: P ∋ v ⟺ v = αu₁+βu₂의 연립방정식 풀기로 전환
02
기술통계 — 분산 · 공분산 · 상관계수
원본 참조: 문제 2 [①②③]
[응용] 2-A ⭐⭐ 보통 수치 변형 (음의 상관)
아래 N = 4개의 데이터 쌍 (xᵢ, yᵢ)가 주어졌다.

i 1 2 3 4
xᵢ 1 2 3 4
yᵢ 4 2 1 1
(1) 표본 평균 x̄, ȳ를 구하라.
(2) 분산 σ²ₓ, σ²_y를 구하라.
(3) 공분산 Cov(X, Y)를 구하라.
(4) 피어슨 상관계수 r을 구하고, 두 변수의 선형 관계를 해석하라.
▼
▶ 정답 및 풀이
(1) 표본 평균
x̄ = (1+2+3+4)/4 = 10/4 = 2.5 ȳ = (4+2+1+1)/4 = 8/4 = 2.0
(2) 분산
σ²ₓ = [(1−2.5)²+(2−2.5)²+(3−2.5)²+(4−2.5)²]/4 = [2.25 + 0.25 + 0.25 + 2.25]/4 = 5/4 = 1.25 편차y: (4−2)=2, (2−2)=0, (1−2)=−1, (1−2)=−1 σ²_y = [4 + 0 + 1 + 1]/4 = 6/4 = 1.5
(3) 공분산
편차곱: (−1.5)(2)+(−0.5)(0)+(0.5)(−1)+(1.5)(−1) = −3 + 0 − 0.5 − 1.5 = −5 Cov(X,Y) = −5/4 = −1.25
(4) 상관계수
σₓ = √1.25 ≈ 1.118, σ_y = √1.5 ≈ 1.225 r = −1.25 / (1.118 × 1.225) ≈ −1.25 / 1.370 ≈ −0.912 → r ≈ −0.91로 강한 음의 선형 상관관계: x가 증가할수록 y는 감소하는 경향이 뚜렷하다.
📌 원본 대비 변형 포인트
원본: xᵢ = {1,2,3,4}, yᵢ = {0,3,2,3} → 양의 상관 (r ≈ +0.775)
응용: yᵢ = {4,2,1,1}으로 변경 → 음의 상관 구조로 전환
핵심 변형: 공분산 부호가 양 → 음으로 바뀌어, 상관계수 해석 방향이 달라짐
[응용] 2-B ⭐⭐ 보통 조건 변형 (N=5, r≈0)
아래 N = 5개의 데이터 쌍이 주어졌다.

i 1 2 3 4 5
xᵢ 1 2 3 4 5
yᵢ 2 4 1 3 5
(1) x̄, ȳ를 구하라.
(2) σ²ₓ, σ²_y를 구하라.
(3) Cov(X, Y)를 구하라.
(4) r을 구하고 두 변수의 관계를 해석하라. 결과가 의미하는 바를 직관적으로 설명하라.
▼
▶ 정답 및 풀이
(1) 표본 평균
x̄ = (1+2+3+4+5)/5 = 15/5 = 3 ȳ = (2+4+1+3+5)/5 = 15/5 = 3
(2) 분산
편차x: −2, −1, 0, 1, 2 σ²ₓ = (4+1+0+1+4)/5 = 10/5 = 2 편차y: −1, 1, −2, 0, 2 σ²_y = (1+1+4+0+4)/5 = 10/5 = 2
(3) 공분산
편차곱: (−2)(−1)+(−1)(1)+(0)(−2)+(1)(0)+(2)(2) = 2 − 1 + 0 + 0 + 4 = 5 Cov(X,Y) = 5/5 = 1
(4) 상관계수
σₓ = σ_y = √2 ≈ 1.414 r = 1 / (√2 × √2) = 1/2 = 0.5 → r = 0.5로 중간 정도의 양의 선형 상관. x가 1 증가할 때 y가 일관되게 증가하지는 않지만, 전반적으로 양의 방향성을 가진다.
📌 원본 대비 변형 포인트
원본: N=4, 양의 상관 (r ≈ 0.775)
응용: N=5로 확장, y값 순서를 섞어 r=0.5로 약화
추가 학습 포인트: N이 달라져도 공식(1/N)은 동일하게 적용됨을 확인
03
이항 분포 · 정규 근사 · MLE
원본 참조: 문제 3 [①②③]
[응용] 3-A ⭐⭐ 보통 수치 변형 (p=0.6)
앞면이 나올 확률이 p = 0.6인 동전을 n = 100번 던진다.
앞면 횟수를 X ~ B(100, 0.6)라 하자.
참고: Φ(−2.04) ≈ 0.0207, Φ(2.04) ≈ 0.9793

(1) E[X]와 Var(X)를 구하라.
(2) 정규 근사로 P(X ≤ 48)을 구하라. Z-score 계산 과정을 명시하라.
(3) 동전 100번 중 앞면이 55번 나왔을 때 MLE p̂를 구하라.
(3-a) 로그 우도 함수 ℓ(p)를 써라.
(3-b) dℓ/dp = 0을 풀어 p̂를 구하라.
(3-c) MLE 결과의 직관적 의미를 서술하라.
▼
▶ 정답 및 풀이
(1) 기댓값과 분산
E[X] = np = 100 × 0.6 = 60 Var(X) = np(1−p) = 100 × 0.6 × 0.4 = 24 → 표준편차 σ = √24 ≈ 4.899
(2) P(X ≤ 48) 정규 근사
Z = (X − np) / √(np(1−p)) Z = (48 − 60) / √24 = −12 / 4.899 ≈ −2.449 ≈ −2.45 P(X ≤ 48) = P(Z ≤ −2.45) ≈ Φ(−2.45) ≈ 0.0071 ※ 문제에 Φ(−2.04)≈0.0207이 주어진 경우, Z≈−2.45는 더 극단적이므로 확률이 약 0.007로 더 작음.
(3) MLE (관측: 55/100)
(3-a) ℓ(p) = 55·log p + 45·log(1−p) (3-b) dℓ/dp = 55/p − 45/(1−p) = 0 55(1−p) = 45p 55 − 55p = 45p 55 = 100p p̂ = 55/100 = 0.55 (3-c) MLE = 관측 빈도 = 55/100 → MLE는 관측된 성공 빈도와 항상 일치한다. 이는 베르누이 시행에서 데이터가 말해주는 가장 자연스러운 확률 추정값이다.
📌 원본 대비 변형 포인트
원본: p=0.8, 관측 X=70 → p̂=0.7, P(X≤70) 계산
응용: p=0.6으로 변경 (Var 증가: 16→24), 관측 X=55 → p̂=0.55
분기점: Var(X)=np(1−p)는 p=0.5에서 최대 → p=0.6이 p=0.8보다 분산이 크다
[응용] 3-B ⭐⭐⭐ 어려움 역방향 문제
X ~ B(100, p)에서 정규 근사 Z = (X − np)/√(np(1−p)) ≈ N(0,1)를 사용한다.

(1) p = 0.5, n = 100일 때, P(X ≤ k) ≈ 0.0228을 만족하는 k를 구하라.
   (힌트: Φ(−2) ≈ 0.0228)
(2) 같은 설정(p=0.5, n=100)에서 P(X ≥ m) = 0.0228을 만족하는 m을 구하라.
(3) 동전 100번 중 앞면이 k번 나왔다면, MLE p̂를 구하고 참값 p=0.5와 비교하라.
▼
▶ 정답 및 풀이
(1) k 역산
E[X] = 100×0.5 = 50, Var(X) = 100×0.5×0.5 = 25, σ = 5 P(X ≤ k) = 0.0228 = Φ(−2) Z = (k − 50)/5 = −2 k − 50 = −10 k = 40
(2) m 역산
P(X ≥ m) = 0.0228 = 1 − P(X ≤ m−1) = 0.0228 → P(X ≤ m−1) = 0.9772 = Φ(2) Z = (m−1 − 50)/5 = 2 m−1 = 60 → m = 61 따라서 P(X ≥ 61) ≈ 0.0228
(3) MLE와 비교
k=40번 관측 시: p̂ = 40/100 = 0.40 참값 p = 0.5 대비 0.10 차이 이처럼 정규 분포에서 2σ 이상 벗어난 관측값은 약 2.3% 확률로 발생하며, MLE는 이런 극단적 샘플에서도 단순히 관측 빈도를 반환함.
📌 원본 대비 변형 포인트
원본: p, n, X가 주어지고 → Z-score와 확률을 출력
응용: 역방향 — 확률값 Φ(−2)≈0.0228이 주어지고 → X값(k, m)을 역산
추가 난이도: P(X ≥ m) = 0.0228에서 대칭성을 이용해야 하는 2단계 역산
04
로지스틱 회귀 — Chain Rule · 경사하강법
원본 참조: 문제 4 [①②③]
[응용] 4-A ⭐⭐ 보통 데이터 변형 + α=0.5
아래 N = 4개의 데이터가 주어졌다.

i 1 2 3 4
xᵢ 1 2 3 4
yᵢ 1 1 0 1
모델: ŷᵢ = σ(zᵢ), zᵢ = θ₀ + θ₁xᵢ. 초기값: θ₀=0, θ₁=0. 학습률 α=0.5.

(1) 초기값에서 모든 ŷᵢ와 오차 ŷᵢ−yᵢ를 구하라.
(2) 전체 손실 J를 계산하라.
(3) ∂J/∂θ₀, ∂J/∂θ₁을 수치적으로 계산하라.
(4) α=0.5로 한 번 경사하강 후 θ₀⁽¹⁾, θ₁⁽¹⁾를 구하라.
(5) 업데이트된 파라미터로 x=2에서 예측 확률을 계산하라.
▼
▶ 정답 및 풀이
(1) 초기 예측값과 오차
θ₀=0, θ₁=0 → zᵢ = 0 for all i → σ(0) = 0.5 i xᵢ yᵢ ŷᵢ ŷᵢ−yᵢ 1 1 1 0.5 −0.5 2 2 1 0.5 −0.5 3 3 0 0.5 +0.5 4 4 1 0.5 −0.5
(2) 손실 J
L₃ = −log(1−0.5) = log2, 나머지 Lᵢ = −log(0.5) = log2 J = (1/4)(log2 + log2 + log2 + log2) = log2 ≈ 0.693
(3) 그래디언트
∂J/∂θ₀ = (1/4)[(−0.5)+(−0.5)+(+0.5)+(−0.5)] = (1/4)(−1) = −0.25 ∂J/∂θ₁ = (1/4)[1·(−0.5)+2·(−0.5)+3·(+0.5)+4·(−0.5)] = (1/4)[−0.5−1+1.5−2] = (1/4)(−2) = −0.5
(4) α=0.5 경사하강
θ₀⁽¹⁾ = 0 − 0.5×(−0.25) = 0 + 0.125 = 0.125 θ₁⁽¹⁾ = 0 − 0.5×(−0.5) = 0 + 0.25 = 0.25
(5) x=2 예측
z = 0.125 + 0.25×2 = 0.125 + 0.5 = 0.625 σ(0.625) ≈ 1/(1+e^{−0.625}) ≈ 1/(1+0.535) ≈ 0.651 → P(y=1|x=2) ≈ 0.651 (원본 α=1일 때보다 업데이트 폭이 작아 0.5에 가까운 예측값)
📌 원본 대비 변형 포인트
원본: yᵢ={0,1,1,1}, α=1 → θ₀⁽¹⁾=0.25, θ₁⁽¹⁾=0.875
응용: yᵢ={1,1,0,1}로 y₁↔y₃ 교환 + α=0.5로 절반

가우시안 MLE — 두 파라미터 동시 추정
원본 참조: 문제 6 [④ 고난도]
[응용] 6-A ⭐⭐⭐ 어려움 데이터셋 변형
N=5개의 데이터 {2, 4, 6, 8, 5}가 N(μ, σ²)에서 생성되었다.

(1) 우도 함수 L(μ, σ²)를 곱 형태로 써라.
(2) ℓ(μ, σ²) = log L을 세 항으로 정리하라.
(3) ∂ℓ/∂μ = 0으로 μ̂ = x̄임을 보이고 값을 구하라.
(4) v = σ²로 치환하여 ∂ℓ/∂v = 0을 풀고 σ̂²을 구하라.
(5) μ̂, σ̂², σ̂를 수치로 구하라.
(6) 불편 추정량 s²을 구하고 σ̂²과 비교하라.
▼
▶ 정답 및 풀이
(1) 우도 함수
L(μ,σ²) = ∏ᵢ₌₁⁵ (1/√(2πσ²)) exp(−(xᵢ−μ)²/(2σ²))
(2) 로그 우도 전개
ℓ(μ,σ²) = −(N/2)log(2π) − (N/2)log(σ²) − (1/(2σ²))Σ(xᵢ−μ)² = −(5/2)log(2π) − (5/2)log(σ²) − (1/(2σ²))Σ(xᵢ−μ)²
(3) μ̂ = x̄ 유도
∂ℓ/∂μ = (1/σ²)Σ(xᵢ−μ) = 0 → Σxᵢ = Nμ → μ̂ = (1/N)Σxᵢ = x̄ 수치: x̄ = (2+4+6+8+5)/5 = 25/5 = 5
(4) σ̂² 유도
v = σ², ∂ℓ/∂v = −N/(2v) + Σ(xᵢ−μ̂)²/(2v²) = 0 → N·v = Σ(xᵢ−μ̂)² → σ̂² = (1/N)Σ(xᵢ−μ̂)²
(5) 수치 계산
편차: (2−5)=−3, (4−5)=−1, (6−5)=1, (8−5)=3, (5−5)=0 편차²: 9, 1, 1, 9, 0 → 합 = 20 σ̂² = 20/5 = 4.0 σ̂ = √4 = 2.0 μ̂ = 5
(6) 불편 추정량 비교
s² = 20/(5−1) = 20/4 = 5.0 σ̂² = 4.0 < s² = 5.0 차이: s² − σ̂² = 1.0 (비율: 25% 과소추정) N−1로 나누는 이유: μ를 모르고 x̄로 추정할 때 자유도가 1 감소하여, N으로 나누면 σ²을 체계적으로 과소추정(편향)하기 때문이다.
📌 원본 대비 변형 포인트
원본: {1,3,5,7,4} → μ̂=4, σ̂²=4, σ̂=2, s²=5
응용: {2,4,6,8,5}로 모든 값을 +1 이동 → μ̂=5, 편차 구조는 동일
σ̂²=4, s²=5로 동일한 수치가 나와 원본과 비교하여 "이동 불변성" 확인 가능
[응용] 6-B ⭐⭐⭐ 어려움 N=6 확장 + 편향 심화
N=6개의 데이터 {1, 3, 5, 7, 4, 4}가 N(μ, σ²)에서 생성되었다.

(1) μ̂ = x̄을 구하라.
(2) σ̂²(MLE)를 구하라.
(3) 불편 추정량 s²을 구하라.
(4) σ̂²과 s²의 비율을 구하고, 이 비율이 N에만 의존함을 식으로 보여라.
(5) N이 커질수록 σ̂²와 s²의 차이가 어떻게 변하는지 설명하라.
▼
▶ 정답 및 풀이
(1) μ̂
x̄ = (1+3+5+7+4+4)/6 = 24/6 = 4
(2) σ̂² (MLE)
편차: −3, −1, 1, 3, 0, 0 편차²: 9, 1, 1, 9, 0, 0 → 합 = 20 σ̂² = 20/6 ≈ 3.333
(3) 불편 추정량 s²
s² = 20/(6−1) = 20/5 = 4.0
(4) 비율의 N 의존성
σ̂² = (1/N)Σ(xᵢ−x̄)² s² = (1/(N−1))Σ(xᵢ−x̄)² σ̂² / s² = (N−1)/N N=5: σ̂²/s² = 4/5 = 0.80 (즉, MLE는 s²의 80%) N=6: σ̂²/s² = 5/6 ≈ 0.833 (즉, MLE는 s²의 83.3%) → 비율 (N−1)/N은 N에만 의존하며 데이터와 무관
(5) N → ∞ 극한
lim_{N→∞} (N−1)/N = 1 N이 커질수록 σ̂²/s² → 1, 즉 MLE 편향이 0에 수렴. N=5: 편향 20%, N=10: 편향 10%, N=100: 편향 1% → 대규모 데이터에서 MLE 편향은 무시 가능해진다.
📌 원본 대비 변형 포인트
원본: N=5 → s²−σ̂²= 5.0−4.0=1.0, 비율 (N−1)/N = 0.8
응용: N=6으로 확장 (데이터에 4를 2개 추가하여 μ̂=4 유지)
추가 심화: (N−1)/N 비율 공식의 N 의존성과 점근 거동(N→∞)까지 분석
