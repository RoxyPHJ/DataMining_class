4.5 편미분 & Gradient
편미분과 그래디언트 벡터, 연쇄 법칙의 수학적 정의와 기계학습 경사하강법으로의 연결
컴퓨터공학부 · 김인수 슬라이드 1–6 편미분 · ∇f · Chain Rule
01 편미분이란? (Partial Derivative)
편미분: 여러 변수가 있을 때, 나머지는 상수로 고정하고 딱 하나의 변수에 대해서만 미분하는 것
직관적 비유 — 산의 높이 h(x, y)

산의 높이를 두 변수 x(동서), y(남북)로 표현할 때:

  • ∂h/∂x → 동쪽으로 걸을 때 기울기
  • ∂h/∂y → 북쪽으로 걸을 때 기울기
→ 어느 방향이 가장 가파른가? 이것이 그래디언트로 이어진다
수식 정의 예시

f(x, y) = x² + 3xy + y² 일 때

∂f/∂x = 2x + 3y ← y는 상수 취급 ∂f/∂y = 3x + 2y ← x는 상수 취급

표기 주의: d 대신 ∂ (라운드 d) 사용 = '편미분 기호'

일반 미분 f(x)

변수가 x 하나
→ df/dx로 미분

편미분 ∂f/∂x

변수가 여러 개
→ 나머지는 묶어두고 x만 미분

표기 주의

d 대신 ∂ (라운드 d)
→ '편미분 기호'

편미분의 기하학적 의미 — 3D 곡면의 단면 기울기

3D 함수 f(x,y)의 곡면 위 한 점에서:

∂f/∂x: y를 고정한 채 x방향으로 자른 단면의 기울기 (파란 선)

∂f/∂y: x를 고정한 채 y방향으로 자른 단면의 기울기 (초록 선)

두 기울기를 합치면 그래디언트 벡터가 된다.

⭐ 기계학습 연결: 손실함수 J(θ₀, θ₁, …)는 파라미터가 여러 개 → 각 파라미터에 대해 편미분해서 어느 방향으로 업데이트할지 결정
02 편미분 — 계산 규칙과 연습
규칙: 미분하는 변수 이외의 것은 전부 숫자(상수)처럼 취급한다. 일반 미분 규칙 그대로 적용.
예제 1  |  f(x, y) = 3x²y + 2y³
∂f/∂x = 6xy
y는 상수 → 3y가 계수
∂f/∂y = 3x² + 6y²
x는 상수 → 3x²는 그대로
예제 2  |  f(x, y) = e^(xy)
∂f/∂x = y · e^(xy)
chain rule, y는 상수
∂f/∂y = x · e^(xy)
chain rule, x는 상수
예제 3  |  f(x, y) = log(x) + xy²
∂f/∂x = 1/x + y²
log(x)는 1/x, y²은 x에 대한 계수
∂f/∂y = 2xy
log(x)는 y 없으니 0, xy²을 y로 미분
⚠ 자주 하는 실수: ∂/∂x [xy²]에서 y²을 그냥 미분하면 안 됨. x에 대한 미분이므로 y²은 계수 → 결과는 y²
03 그래디언트 벡터 (Gradient Vector) ∇f
그래디언트: 모든 변수에 대한 편미분을 하나의 벡터로 묶은 것 — '함수가 가장 가파르게 증가하는 방향'을 가리킨다
정의

f(θ₁, θ₂, …, θₙ)에 대해

∇f = [∂f/∂θ₁, ∂f/∂θ₂, …, ∂f/∂θₙ]ᵀ 각 원소 = 해당 변수 방향의 기울기
예시

J(θ₁, θ₂) = θ₁² + 2θ₂²

∇J = [∂J/∂θ₁, ∂J/∂θ₂]ᵀ = [2θ₁, 4θ₂]ᵀ
📍 방향

∇f는 현재 위치에서
함수값이 가장 빠르게 증가하는 방향

⬇ −∇f 방향으로 이동

그래디언트의 반대 방향
→ 가장 가파르게 감소하는 방향

🔁 경사하강법과 연결
θ ← θ − α · ∇J(θ)

그래디언트 반대 방향으로 조금씩 이동 = 최솟값 탐색

그래디언트 벡터 방향 시각화 — J(θ₁, θ₂) = θ₁² + 2θ₂²

등고선(contour) 위 각 점에서 ∇J (빨간 화살표)는 함수가 가장 빠르게 증가하는 방향을 가리킨다.

−∇J (초록 화살표)는 그 반대 방향으로, 경사하강법이 이동하는 방향이다.

중심 별표가 전역 최솟값 (0, 0)이다.

⭐ 핵심 직관: 경사하강법은 '현재 위치에서 그래디언트를 계산하고, 그 반대 방향으로 α만큼 이동'하는 과정을 반복하는 것
04 연쇄 법칙 (Chain Rule) — 합성함수의 미분
연쇄 법칙: 함수 안에 함수가 겹쳐 있을 때 미분하는 규칙. 딥러닝 역전파(Backpropagation)의 핵심 수학 도구.
① 1변수 연쇄 법칙
y = f(u), u = g(x)
dy/dx = dy/du · du/dx
예) y = (3x + 1)² 이면  →  u = 3x+1, y = u²  →  dy/dx = 2u · 3 = 6(3x+1)
② 다변수 연쇄 법칙 (기계학습에서 핵심!)
L = f(ŷ),   ŷ = σ(z),   z = θᵀx
∂L/∂θ = ∂L/∂ŷ · ∂ŷ/∂z · ∂z/∂θ
∂L/∂ŷ

loss가 예측값에 얼마나 민감한가

∂ŷ/∂z

σ(z) 미분 = σ(1 − σ)

∂z/∂θ

z = θᵀx → 결과는 x

역전파(Backpropagation) = 연쇄 법칙의 반복 적용
⭐ 역전파(Backpropagation) = 연쇄 법칙을 출력층에서 입력층 방향으로 반복 적용하는 것.
각 층의 그래디언트를 계산할 때 '앞 층에서 이미 계산된 ∂L/∂(앞 출력)'을 재사용 → 효율적
05 편미분 · 그래디언트 — 기계학습 핵심 요약
개념 비교 요약표 (슬라이드 6)
개념 수식 기계학습에서의 역할
편미분
∂f/∂θₖ		 ∂J/∂θₖ 파라미터 θₖ 하나를 바꿀 때 손실 J가 얼마나 변하는가
그래디언트
∇J		 ∇J = [∂J/∂θ₁ … ∂J/∂θₙ]ᵀ 모든 파라미터에 대한 기울기 벡터 → 이동 방향 결정
경사하강법 θ ← θ − α · ∇J(θ) 그래디언트 반대 방향으로 이동해 J를 최소화
연쇄 법칙 ∂L/∂θ = ∂L/∂ŷ · ∂ŷ/∂z · ∂z/∂θ 여러 층 합성 시 그래디언트 계산 → 역전파의 수학적 근거
전체 학습 흐름 다이어그램
핵심 공식 모음

편미분 (x에 대해)

∂f/∂x ← y, z, ... 는 상수 취급

그래디언트 벡터

∇f = [∂f/∂θ₁, ∂f/∂θ₂, ..., ∂f/∂θₙ]ᵀ

경사하강법 업데이트

θ ← θ − α · ∇J(θ)

다변수 연쇄 법칙 (역전파)

∂L/∂θ = ∂L/∂ŷ · ∂ŷ/∂z · ∂z/∂θ
