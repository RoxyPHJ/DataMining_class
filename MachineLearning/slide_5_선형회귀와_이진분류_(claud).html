01

선형 회귀 (Linear Regression)

회귀(Regression)란?

연속적인 수치(실수)를 예측하는 지도 학습 문제. 입력 특징 벡터 x로부터 실수값 y를 예측한다.

주택 가격 예측

입력 x: 면적 · 위치 · 방 수
출력 y: 가격 (만원)

주가 예측

입력 x: 전일 종가 · 거래량
출력 y: 다음 날 종가

기온 예측

입력 x: 습도 · 기압 · 계절
출력 y: 최고 기온 (°C)

전력 수요 예측

입력 x: 요일 · 시간대 · 온도
출력 y: 전력 소비 (MW)

선형 회귀 모델
ŷ = θ₀ + θ₁x ← 단순 선형 (입력 1개)
ŷ = θᵀx ← 벡터 표현 (bias 항 x₀=1 포함)
또는 ŷ = Wx + b
🎯
학습 목표: 데이터 (x₁, y₁), …, (xₙ, yₙ)가 주어졌을 때, 예측값 ŷ와 실제값 y의 차이를 가장 작게 만드는 θ를 찾는 것.

선형 회귀의 기하학적 의미

산점도의 데이터 포인트들에 가장 잘 맞는 직선을 찾는 과정.

각 데이터 포인트에서 회귀 직선까지의 수직 거리(잔차)의 제곱합을 최소화 = 최소제곱법(Least Squares).

잔차 eᵢ = yᵢ − ŷᵢ = yᵢ − θᵀxᵢ
01a

MSE 비용 함수

평균 제곱 오차 (MSE, Mean Squared Error)
J(θ) = (1/N) Σᵢ (ŷᵢ − yᵢ)²
     = (1/N) Σᵢ (θᵀxᵢ − yᵢ)²

학습 목표: θ* = argminθ J(θ)

왜 제곱인가?

  • 음수 오차와 양수 오차가 상쇄되지 않도록
  • 절댓값 대신 제곱으로 모두 양수화
  • 미분이 가능해 경사하강법 적용 가능

왜 평균인가?

  • 데이터 수 N에 무관하게 비용 함수 스케일 유지
  • 데이터가 늘어도 값이 폭발하지 않음
  • 배치 크기가 달라도 비교 가능

MSE의 기하학적 의미

  • 각 데이터 포인트에서 회귀직선까지의 수직 거리의 제곱합 최소화
  • = 최소제곱법 (Least Squares Method)

MSE vs MAE 비교

MSE: 이상치에 민감 (오류가 제곱으로 커짐), 미분이 연속적이라 최적화 용이. 딥러닝의 기본.

MAE: 이상치에 더 강인하지만, 미분 불연속 지점 존재 (z=0에서).

MAE = (1/N) Σ|ŷᵢ − yᵢ|
MSE = (1/N) Σ(ŷᵢ − yᵢ)²
01b

정규방정식 (Normal Equation)

MSE를 미분을 이용해 해석적으로 θ 구하기
Step 1
행렬 표현: J(θ) = (1/N)||Xθ − y||²
X: N×d 설계 행렬 (각 행이 하나의 샘플),   y: N×1 레이블 벡터
Step 2
미분: ∇θJ(θ) = (2/N) XᵀXθ − (2/N) Xᵀy = 0
→ XᵀXθ = Xᵀy
Step 3
해: θ = (XᵀX)⁻¹ Xᵀy ← 정규방정식 (Normal Equation)

정규방정식의 장점

  • 반복 학습 불필요 — 한 번에 해를 구함
  • 하이퍼파라미터(학습률) 없음
  • 소규모 데이터에서 매우 효율적

정규방정식의 단점

  • (XᵀX) 역행렬 계산: O(d³) — 특징 수 d가 크면 매우 느림
  • XᵀX가 특이행렬이면 역행렬 불가 → Ridge 정규화로 해결
  • Rank가 낮을 경우 (d > N)
  • 딥러닝(비선형 모델)에는 적용 불가 → 경사하강법 사용
01c

경사하강법 (Gradient Descent)

파라미터 업데이트 규칙
θ ← θ − α · ∇θJ(θ) α: 학습률 (learning rate)

MSE에 대한 그래디언트 (선형 회귀):
∇θJ(θ) = (2/N) Xᵀ(Xθ − y)

→ θ ← θ − (2α/N) Xᵀ(Xθ − y)

경사하강법의 원리

비용 함수 J(θ)의 기울기(gradient) 반대 방향으로 θ를 반복 이동시켜 최솟값을 찾는다.

학습률 α가 너무 크면 발산, 너무 작으면 학습이 매우 느림.

⚠️
학습률 스케줄링 또는 Adam · RMSProp 같은 적응적 최적화 기법 사용 권장

배치 경사하강법 (Batch GD)

매 스텝에 전체 N개 데이터 사용
→ 안정적이지만 느림
→ 대규모 데이터에 비효율적

확률적 경사하강법 (SGD)

매 스텝에 1개 샘플 사용
→ 빠르지만 불안정 (노이즈 큼)
→ 지역 최솟값 탈출 가능

미니배치 경사하강법 ★

매 스텝에 B개 샘플 (B=32~256)
→ 배치와 SGD의 절충안
→ 딥러닝의 표준 방식

01d

다항 회귀 & 과적합

다항 특징 변환

비선형 관계도 '특징 변환'으로 선형 회귀 틀 안에서 표현할 수 있다.

원래 특징 x → [1, x, x², x³, …, xᵐ] 으로 확장 후 선형 회귀 적용

1차 (Linear) — Underfitting

모델이 너무 단순
데이터의 패턴을 담지 못함

High Bias

적절한 차수 — Good Fit ✓

적절한 복잡도
훈련·테스트 오차 균형

Bias-Variance 균형

고차 (n차) — Overfitting

모델이 너무 복잡
훈련 데이터의 노이즈까지 학습

High Variance

🔗
이전 강의 연결: Underfitting = High Bias / Overfitting = High Variance → Bias-Variance Tradeoff
해결책: 교차 검증(Cross-Validation)으로 차수 선택 / 정규화(Regularization)로 계수를 제한
02

이진 분류 (Binary Classification)

분류(Classification)란?

입력 x를 유한한 클래스 중 하나로 할당하는 지도 학습 문제.

회귀 vs 분류

회귀: y ∈ ℝ (연속 실수)
예) 가격, 온도, 주가


분류: y ∈ {0, 1, …, K−1} (이산 클래스)
이진 분류: y ∈ {0, 1}

스팸 메일 분류

스팸 (1) / 정상 (0)

질병 진단

양성 (1) / 음성 (0)

신용카드 부정거래

사기 (1) / 정상 (0)

왜 MSE로 분류 문제를 풀면 안 되는가?
❌
선형 회귀는 ŷ가 [0,1] 범위를 벗어날 수 있다 → '확률'로 해석 불가
❌
이상치 1개가 결정 경계를 크게 왜곡시킨다 (MSE가 큰 오차를 제곱으로 과중 처벌)
❌
MSE는 오차 분포가 Gaussian임을 가정 → 이진 레이블 {0,1}에는 부적합
02a

시그모이드 (Sigmoid) 함수

Sigmoid 함수 정의
σ(z) = 1 / (1 + e⁻ᶻ)    출력 범위: (0, 1)

σ(0) = 0.5

z = 0이면 출력 = 0.5 (중립)

z → +∞

σ(z) → 1 (C₁에 가까워짐)

z → −∞

σ(z) → 0 (C₀에 가까워짐)

미분 공식 (역전파 핵심!)

σ'(z) = σ(z) · (1 − σ(z))

S자 형태의 곡선

z축 기준 주요 값:

z ≪ 0 → σ ≈ 0
z = −2 → σ ≈ 0.12
z = 0 → σ = 0.50
z = 2 → σ ≈ 0.88
z ≫ 0 → σ ≈ 1
로지스틱 회귀 모델
P(y=1|x; θ) = σ(θᵀx)
              = 1 / (1 + exp(−θᵀx))
03

Cross-Entropy 손실 & 경사하강법

Binary Cross-Entropy (단일 샘플)
L(ŷ, y) = −[y · log(ŷ) + (1 − y) · log(1 − ŷ)]
단, ŷ = σ(θᵀx) = P(y=1|x)

y = 1 일 때

L = −log(ŷ)

ŷ = 1이면 L = 0,   ŷ → 0이면 L → ∞
예측이 1에 가까울수록 손실 감소

y = 0 일 때

L = −log(1 − ŷ)

ŷ = 0이면 L = 0,   ŷ → 1이면 L → ∞
예측이 0에 가까울수록 손실 감소

전체 데이터셋 손실 (N개 샘플 평균)
J(θ) = −(1/N) Σᵢ [yᵢ · log(ŷᵢ) + (1 − yᵢ) · log(1 − ŷᵢ)]
🔬
Cross-Entropy 손실 최소화 = Bernoulli 분포에 대한 MLE (최대 우도 추정)와 수학적으로 완전히 동치 — MLE 수업에서 증명 예정
로지스틱 회귀 경사하강법 — 그래디언트 도출
Step 1
J(θ) = −(1/N) Σ [y · log(σ(z)) + (1 − y) · log(1 − σ(z))] 에서 z = θᵀx
Step 2
∂J/∂z = σ(z) − y   (시그모이드 미분 성질 σ'(z) = σ(z)(1 − σ(z)) 이용)
Step 3
∂J/∂θ = (σ(z) − y) · x → (1/N) Xᵀ(ŷ − y) ← MSE Gradient와 형태 동일!
파라미터 업데이트:
θ ← θ − (α/N) · Xᵀ(ŷ − y)    ( ŷ = σ(Xθ) )
✅
로지스틱 회귀는 정규방정식 없음. Sigmoid를 포함하므로 해석적 해가 없다 → 반드시 반복적 최적화(경사하강법) 사용.
단, MSE와 달리 Cross-Entropy + Sigmoid 조합은 볼록(convex) 함수 → 지역 최솟값이 곧 전역 최솟값.
03a

결정 경계 (Decision Boundary)

결정 규칙
P(y=1|x) = σ(θᵀx) ≥ 0.5 → y = 1로 예측
P(y=1|x) = σ(θᵀx) < 0.5 → y = 0으로 예측

결정 경계 조건:
σ(θᵀx) = 0.5 ⟺ θᵀx = 0 → θ₀ + θ₁x₁ + θ₂x₂ = 0 (2차원 입력)

2차원 예시

θ = [−6, 1, 1] (θ₀=−6, θ₁=1, θ₂=1) 이면:

결정 경계: x₁ + x₂ = 6 (직선)

y = 1 영역

x₁ + x₂ > 6

y = 0 영역

x₁ + x₂ < 6

💡
비선형 결정 경계: 입력 특징을 다항 변환하면 곡선 경계 가능 (원형, 타원, 쌍곡선 등)
04

다중 클래스 확장 & 정규화

Softmax 함수 (다중 분류)

이진 분류(Sigmoid)를 K개 클래스로 일반화 → Softmax 함수

P(y=k | x; W) = exp(wₖᵀx) / Σⱼ exp(wⱼᵀx) → K개 클래스의 확률 합 = 1

Sigmoid (이진 분류)

  • 클래스 K = 2
  • 출력: P(y=1|x) ∈ (0,1) (스칼라)
  • σ(z) = 1/(1 + e⁻ᶻ)
  • Softmax(K=2)와 동치

Softmax (다중 분류)

  • 클래스 K ≥ 2
  • 출력: [P(y=0), …, P(y=K−1)] 벡터
  • 각 원소 ∈ (0,1), 합 = 1
  • → 확률 분포 형태
다중 클래스 Cross-Entropy (Categorical CE)
J(W) = −(1/N) Σᵢ Σₖ yᵢₖ · log P(y=k|xᵢ; W)
yᵢₖ: one-hot 레이블
🔬
Categorical Cross-Entropy 최소화 = Categorical 분포에 대한 MLE — MLE 수업에서 증명
정규화 (Regularization)
과적합(Overfitting): 훈련 데이터의 노이즈까지 학습 → 새로운 데이터에서 성능 저하

정규화: 손실 함수에 패널티 항을 추가해 θ가 과도하게 커지는 것을 제한

L2 정규화 (Ridge)

J(θ) = 원래 손실 + λ||θ||²
  • θ를 0 방향으로 축소 (shrinkage)
  • 모든 특징을 조금씩 사용
  • = Gaussian Prior를 사용한 MAP 추정

L1 정규화 (Lasso)

J(θ) = 원래 손실 + λ||θ||₁
  • 일부 θ를 정확히 0으로 만듦
  • 자동 특징 선택 (sparse solution)
  • = Laplace Prior를 사용한 MAP 추정
정규화 강도 λ (하이퍼파라미터)
05

선형 회귀 vs 로지스틱 회귀 — 비교 & 핵심 요약

핵심 비교표
비교 항목 선형 회귀 (Linear Reg.) 로지스틱 회귀 (Logistic Reg.)
문제 유형 회귀 (연속 출력) 이진 분류 (확률 출력)
출력 범위 ŷ ∈ ℝ (임의 실수) ŷ ∈ (0, 1) (확률)
출력 함수 ŷ = θᵀx (항등 함수) ŷ = σ(θᵀx) (Sigmoid)
손실 함수 MSE: (1/N)Σ(ŷ − y)² Binary CE: −(1/N)Σ[y·log(ŷ)+(1−y)·log(1−ŷ)]
해석적 해 O 정규방정식 (XᵀX)⁻¹Xᵀy X 반드시 경사하강법
MLE 연결 MSE = Gaussian MLE Cross-Entropy = Bernoulli MLE
⭐
두 모델 모두 '선형 결정 경계'를 가진다. 비선형 경계가 필요하면 특징 변환 또는 신경망을 사용한다.
기계학습 관점 요약

📐 모델

  • 선형: ŷ = θᵀx (= Wx + b)
  • 로지스틱: ŷ = σ(θᵀx)
  • 신경망: ŷ = f(x; θ) (비선형 구성)

📉 손실 함수

  • MSE: (1/N)Σ(ŷᵢ−yᵢ)² [회귀]
  • Binary CE: [이진 분류]
  • Categorical CE: [다중 분류]

⚙️ 최적화

  • 정규방정식: θ=(XᵀX)⁻¹Xᵀy (소규모)
  • 배치 GD: 전체 데이터 사용
  • 미니배치 GD: 딥러닝 표준 ★

🔗 신경망 연결

  • 출력층 = 선형/로지스틱 회귀의 일반화
  • L2 정규화 = Gaussian Prior MAP
  • L1 정규화 = Laplace Prior MAP
