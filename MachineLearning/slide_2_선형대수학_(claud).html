PART 1

벡터 이론

01  벡터의 표현 · 전치 · 크기 · 단위벡터

벡터의 표현 (슬라이드 3)

  • 벡터: 크기와 방향을 가지는 임의의 물리량
  • 패턴 인식에서 인식 대상 객체는 특징 벡터(feature vector)로 표현
  • d차원 공간상의 한 점으로 표현됨
// 열벡터 표현 (d×1 행렬)
x = [x₁, x₂, ..., xd]T  ∈  ℝd
// 예) 키·몸무게·나이 → 3차원 특징 벡터
x = [x₁, x₂, x₃]T

전치 (Transpose, 슬라이드 4)

  • N×1 열벡터 → 1×N 행벡터 (행과 열을 바꿈)
xT = [x₁, x₂, ..., xN] (열→행)
(AB)T = BTAT
(AT)T = A

크기 (Norm)

  • 원점에서 벡터 공간상의 한 점까지의 거리
‖x‖ = √(x₁² + x₂² + ... + xN²)
     = √(xTx)

단위 벡터 (Unit Vector)

  • 크기가 1인 벡터: ‖u‖ = 1
  • 벡터 v ≠ 0 이면 v 방향의 단위벡터: u = v / ‖v‖
  • 벡터를 단위벡터로 만드는 과정 = 정규화(normalization)
02  내적 (Inner Product / Dot Product)

내적의 정의 (슬라이드 5)

  • 같은 차원의 두 벡터 A, B에 대해 대응 성분끼리 곱하여 합산
  • 결과: 실수 스칼라
A · B = A₁B₁ + A₂B₂ + ... + AnBn = ATB

// 기하학적 해석 (각도 θ)
A · B = ‖A‖ ‖B‖ cosθ
// θ = 90° → 내적 = 0 → 두 벡터 직교(orthogonal)
핵심 의미 내적은 한 벡터가 다른 벡터 방향으로 얼마나 '투영'되는지를 측정합니다. 유사도(similarity)·Attention 메커니즘의 수학적 기반입니다.
교환법칙 ✓
A · B = B · A
분배법칙 ✓
A · (B+C) = A·B + A·C
스칼라 ✓
(αA) · B = α(A · B)
자기 내적
A · A = ‖A‖²

내적의 기하학적 의미

  • 벡터 A, B 사이 각도 θ일 때: A · B = ‖A‖‖B‖cosθ
  • θ = 0°: 최대 (같은 방향)
  • θ = 90°: 0 (수직)
  • θ = 180°: 최소 (반대 방향)
03  외적 (Cross Product)

외적의 정의 (슬라이드 6)

  • A, B ∈ ℝ³ 인 3차원 벡터에 대해 정의
  • 결과: 스칼라가 아닌 벡터 (내적과 구별)
// 크기: 평행사변형 면적과 동일
‖A × B‖ = ‖A‖ ‖B‖ sinθ

// 방향: A와 B에 동시에 수직 (오른손 법칙)
// 성분 계산:
A×B = [A₂B₃−A₃B₂,  A₃B₁−A₁B₃,  A₁B₂−A₂B₁]T
반교환법칙 ✗
A×B = −(B×A)
자기 외적
A×A = 0
분배법칙 ✓
A×(B+C) = A×B + A×C
기계학습 활용
법선벡터, 3D 회전

벡터 내적 · 외적 성질 비교 (슬라이드 7)

성질 내적 (Dot Product) 외적 (Cross Product)
결과 형태스칼라벡터 (3D 전용)
교환법칙A·B = B·A  ✓A×B = −B×A  ✗
분배법칙A·(B+C) = A·B + A·C  ✓A×(B+C) = A×B + A×C  ✓
자기 곱A·A = ‖A‖²A×A = 0
기하학적 의미cosθ 투영 / 유사도 측정sinθ 면적 / 수직 방향 탐색
기계학습 활용유사도, 투영, Attention법선벡터, 3D 회전
04  선형 결합 · 선형 종속 · 선형 독립

선형 결합 (Linear Combination, 슬라이드 8)

  • 벡터 집합 {x₁, x₂, ..., xₘ}을 스칼라 계수 {α₁, ..., αₘ}으로 곱하여 합산
y = α₁x₁ + α₂x₂ + ... + αₘxₘ

선형 종속 (Linearly Dependent)

  • 벡터 집합 중 적어도 하나가 나머지의 선형 결합으로 표현 가능
  • 모두 0이 아닌 계수 α₁,...,αₘ 가 존재하여 합 = 0
의미집합 내에 '중복 정보'가 존재함

선형 독립 (Linearly Independent)

  • α₁x₁+...+αₘxₘ = 0 을 만족하는 유일한 해가 α₁=α₂=...=αₘ=0 뿐
  • 어떤 벡터도 다른 벡터들의 선형 결합으로 표현 불가
의미집합 내 모든 벡터가 서로 '독립적인 정보'를 담고 있음
05  기저 집합 (Basis Set)

기저 집합 (슬라이드 9~10)

  • N×1로 표현되는 모든 벡터를 나타낼 수 있는 벡터의 집합
  • 임의의 N×1 벡터 x를 기저 {u₁,...,uₙ}로 표현:
x = a₁u₁ + ... + aₙuₙ
// {a₁,...,aₙ}: 기저 {uᵢ}에 따른 x의 성분
필요충분조건 {u₁,...,uₙ}가 선형 독립

표준 기저 예시

// ℝ²
e₁=[1,0]T,  e₂=[0,1]T

// ℝ³
e₁=[1,0,0]T, e₂=[0,1,0]T, e₃=[0,0,1]T

직교 기저 / 정규직교 기저

직교 기저 (Orthogonal)
uᵢ · uⱼ = 0 (i ≠ j)
정규직교 기저 (Orthonormal)
uᵢ · uⱼ = δᵢⱼ
정규직교 기저의 장점 투영 계산이 단순해짐: aᵢ = x · uᵢ

Gram-Schmidt 정규직교화

  • 임의의 선형 독립 집합 → 정규직교 기저로 변환
  • ① u₁ = v₁ / ‖v₁‖
  • ② 각 vₖ에서 이전 기저들의 투영 성분 제거 후 정규화
기계학습에서의 의미 신경망: 각 층이 데이터를 새로운 기저로 표현 (representation learning).
좋은 기저(특징)를 찾는 것이 딥러닝의 핵심 목표 중 하나.
06  Span과 차원의 확대 · 축소

Span (생성 공간, 슬라이드 11)

  • 벡터 집합 {v₁,...,vₖ}의 모든 가능한 선형 결합들의 집합
span({v₁,...,vₖ}) = { α₁v₁ + ... + αₖvₖ | αᵢ ∈ ℝ }

// 예1) v₁=[1,0]ᵀ, v₂=[0,1]ᵀ → span = ℝ² (2D 평면 전체)
// 예2) v₁=[1,0]ᵀ 만 있으면 span = x축 방향 직선

차원의 확대 (Embedding)

  • 저차원 x ∈ ℝᵈ → 고차원 ℝᴰ의 부분공간을 새로 span
  • → 선형 분리 가능성 증가
예시 SVM 커널 트릭: 원래 공간에서 비선형 경계 → 고차원에서 선형 분리
신경망 은닉층: 각 층이 데이터를 새로운 벡터 공간으로 선형 변환 후 비선형 활성화

차원의 축소 (Dimensionality Reduction)

  • 고차원에서 핵심 정보가 span하는 저차원 부분공간으로 투영
  • 데이터가 실제로 span하는 공간의 유효 차원(rank)을 찾는 것이 핵심
  • 노이즈·중복 특징 제거 → 계산 효율화 및 일반화 성능 향상
07  벡터의 투영 (Projection)

투영의 정의 (슬라이드 12)

  • 벡터 x를 단위벡터 u 방향으로 투영
  • 성분 크기 = x·u
  • 투영 벡터 = (x·u)·u
// 부분공간 S = span({u₁,...,uₖ})로의 투영
proj(x) = UUTx    (U=[u₁...uₖ])

// 잔차 (residual)
r = x − proj(x)    (⊥ S)

투영과 선형 종속의 관계

  • v가 S에 선형 종속이면: 잔차 = 0 → 새 차원 추가 없음
  • v가 S에 선형 독립이면: 잔차 ≠ 0 → 새로운 방향을 span
기저 구성 원리 잔차 ≠ 0인 벡터들만 순차 추가 → Gram-Schmidt 정규직교화

선형 변환 = 새로운 벡터 공간으로의 투영

  • 행렬 W: x → Wx
  • x의 각 성분이 W의 행벡터 방향으로 투영된 크기들의 새 벡터
  • Deep Learning: 각 층의 가중치 행렬 W가 데이터를 새로운 부분공간으로 반복 투영 및 재구성
08  선형 종속과 표현의 축소 / Latent Representation

선형 종속과 표현의 축소 (슬라이드 13)

  • 선형 종속 벡터 집합: rank < 벡터 수 → 실제 span 차원이 더 낮음 → 중복 정보 존재
  • d차원 특징 벡터가 실제로는 k차원 부분공간에 집중 (k << d) → k개의 기저로 표현 가능
// 선형 독립인 k개의 기저 {u₁,...,uₖ}로 x 투영
[x·u₁, ..., x·uₖ] = k차원 압축 표현

Deep Learning의 Latent Representation

Latent Vector z
은닉층 출력 = x의 새로운 벡터 공간 좌표
Latent Space
z가 span하는 공간; 유사 데이터는 가까이 위치
Autoencoder
Encoder x→z (차원 축소) / Decoder z→x' (차원 복원)
Transformer
Attention이 입력 벡터들의 가중 선형 결합 → 새로운 부분공간으로 투영하여 context 표현
PART 2

행렬 대수

09  특수 행렬 종류

전치 행렬 (Transpose, 슬라이드 14)

  • 행렬 A의 행과 열을 바꾼 행렬: (Aᵀ)ᵢⱼ = Aⱼᵢ
(AB)T = BTAT   |   (AT)T = A
행렬 종류 정의 예 / 활용
정방 행렬 (Square) 행 수 = 열 수 (n×n) 역행렬, Rank 정의의 기본
대각 행렬 (Diagonal) D = diag(d₁, d₂, ..., dₙ) 스케일 변환, SVD의 Σ 행렬
항등 행렬 (Identity) 대각=1, 비대각=0: AI = IA = A 역행렬 정의 기준
대칭 행렬 (Symmetric) A = Aᵀ (대각선 기준 대칭) 공분산 행렬, 그람 행렬(GᵀG)
영 행렬 (Zero) 모든 성분 = 0 —
직교 행렬 (Orthogonal) AᵀA = AAᵀ = I (→ A⁻¹ = Aᵀ) 회전 행렬, Householder, QR 분해

직교 행렬 (Orthogonal Matrix, 슬라이드 15)

  • 정방행렬 A가 AᵀA = AAᵀ = I를 만족할 때
  • → A⁻¹ = Aᵀ (역행렬 = 전치행렬 → 계산이 간단해짐)
  • 열벡터(행벡터)들이 서로 직교하는 단위벡터들로 구성
  • 직교 변환은 벡터의 길이와 내적을 보존 (거리·각도 불변)
기계학습 응용 QR 분해, SVD: 정규직교 행렬을 이용한 행렬 분해 → 수치 안정성 높음
정규직교 가중치 초기화: 그래디언트 소실/폭발 방지에 도움
10  행렬의 곱셈

행렬 곱셈 규칙 (슬라이드 16)

  • A (m×k), B (k×n) 일 때 C = AB 는 m×n 행렬
  • Cᵢⱼ = A의 i번째 행벡터와 B의 j번째 열벡터의 내적
Cij = Σl=1..d AilBlj

// 결합법칙 ✓
(AB)C = A(BC)

// 교환법칙 ✗ (일반적으로)
AB ≠ BA

// 전치
(AB)T = BTAT
11  행렬의 계수 (Rank)

Rank의 정의 (슬라이드 17)

  • 행렬 A에서 선형 독립인 열벡터(또는 행벡터)의 최대 수
  • r(A) = rank(A) ≤ min(m, n)  (m×n 행렬)
// 예) 단위행렬 I₃의 세 열벡터 모두 선형 독립
rank(I₃) = 3

비특이 vs 특이 행렬

비특이 (Non-singular) ✓
rank(Aₙₓₙ) = n → 역행렬 존재
특이 (Singular) ✗
rank(Aₙₓₙ) < n → 역행렬 없음

기계학습에서의 의미

  • 데이터 행렬 X의 rank = 데이터가 실제로 span하는 공간의 차원
  • rank가 낮으면 선형 종속 특징이 많아 중복 정보 존재 → 차원 축소 여지
  • 가중치 행렬의 rank = 신경망 층이 실제로 활용하는 독립 방향의 수
12  역행렬 (Inverse Matrix)

역행렬의 정의 (슬라이드 18)

  • 정방행렬 A에 대해 AX = XA = I를 만족하는 X
  • A⁻¹ 로 표기; 역행렬이 존재하려면 det(A) ≠ 0 (비특이 행렬)
(A⁻¹)⁻¹ = A
(AB)⁻¹ = B⁻¹A⁻¹
(AT)⁻¹ = (A⁻¹)T
(αA)⁻¹ = (1/α)A⁻¹  (α ≠ 0)

기계학습에서의 활용

  • 선형 회귀 정규방정식:
θ = (XTX)⁻¹XTy  (최소제곱해)
// XᵀX가 특이행렬이면 → pseudo-inverse 사용
직교 행렬의 이점 A⁻¹ = Aᵀ → 역행렬 계산 불필요 (계산 효율 매우 높음)
의사역행렬 (Pseudo-inverse) XᵀX가 rank 부족으로 역행렬이 존재하지 않을 때 대안으로 사용
13  선형 변환 (Linear Transformation)

선형 변환의 정의 (슬라이드 19~20)

  • 벡터 공간 Xᴺ 에서 벡터 공간 Yᴹ 으로 매핑: y = Wx (W: M×N 행렬)
  • M < N: 차원 축소 (압축 표현)
  • M > N: 차원 확대 (임베딩)
// 정규직교 변환: Aᵀ = A⁻¹이면
‖Ax‖ = ‖x‖   (크기 보존)
(Ax)·(Ay) = x·y  (내적/각도 보존)
// → '회전 변환'에 해당
정규직교 변환의 기계학습 응용 QR 분해, SVD — 수치 안정성 확보
정규직교 가중치 초기화 — 그래디언트 소실/폭발 방지
14  핵심 개념 정리 (슬라이드 21)

전체 핵심 개념 — 딥러닝 연결

개념 핵심 내용 딥러닝 연결
Span 벡터들로 도달 가능한 공간 전체 Latent Space의 구조 정의
선형 독립 어떤 벡터도 나머지의 선형 결합으로 표현 불가 중복 없는 정보 방향
투영 부분공간으로의 최근접 표현 (UUᵀx) 각 층의 선형 변환 (Wx)
Rank 선형 독립 벡터의 최대 수 = span 차원 가중치 행렬의 유효 차원
