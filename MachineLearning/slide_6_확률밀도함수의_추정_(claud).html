01
확률론적 추론 (Probabilistic Inference)
왜 확률로 모델을 설명하는가?
  • 데이터는 노이즈를 포함한다
  • 동일한 입력 x에도 출력 y가 다를 수 있다 → 결정론적 함수로는 불완전
  • 확률 분포 P(y | x ; θ)로 모델링하면 불확실성을 수치화할 수 있다
  • 파라미터 θ를 '데이터를 가장 잘 설명하는 분포'로 결정하는 것이 핵심 과제
모집단 분포

데이터가 실제로 따르는 알 수 없는 분포

p_data(x)
파라미터 θ

분포를 결정하는 변수

평균 μ, 분산 σ², 가중치 W 등
추정 목표

관측 데이터 X로부터 최적의 θ를 찾는 것

핵심 질문: 관측 데이터 X = {x₁, x₂, …, xₙ}가 주어졌을 때, 어떤 θ가 이 데이터를 가장 잘 설명하는가?
MLE θ̂* = argmax P(X | θ)

MAP θ̂* = argmax P(θ | X)
02
최대 우도 추정 (MLE)
우도 함수 (Likelihood Function)

우도 L(θ ; X)는 '파라미터 θ가 고정되었을 때 데이터 X가 나올 확률'이다.

Likelihood L(θ; X) — "X를 봤을 때 θ가 얼마나 그럴듯한가?"
X를 고정하고 θ의 함수로 본다
확률 P(X|θ) — "θ를 알 때 X가 얼마나 자주 일어나나?"
θ를 고정하고 X의 함수로 본다
① 우도 함수 (i.i.d. 가정) L(θ ; X) = P(x₁, x₂, … , xₙ | θ) = ∏ᵢ P(xᵢ | θ)
② 로그 우도 (곱 → 합, 수치 안정성) ℓ(θ) = log L(θ ; X) = Σᵢ log P(xᵢ | θ)
← log는 단조증가 → argmax 보존
③ MLE 추정량 θ_MLE = argmax_θ ℓ(θ)
→ 필요조건: ∇_θ ℓ(θ) = 0
i.i.d.: 데이터 x₁ … xₙ이 동일한 분포에서 독립적으로 샘플링되었다고 가정 → 결합 확률을 곱으로 분리 가능
MLE — Gaussian 분포에 적용

원본 분포가 N(μ, σ²)으로 가정될 때, 파라미터 θ = (μ, σ²)를 MLE로 추정한다.

로그 우도 전개 ℓ(μ, σ²) = −(n/2)·log(2πσ²) − Σ(xᵢ − μ)² / (2σ²)
∂ℓ/∂μ = 0 → μ 추정
∂ℓ/∂μ = Σ(xᵢ − μ) / σ² = 0 ⟹ μ̂_MLE = (1/n) Σxᵢ = x̄
표본 평균이 MLE 추정량
∂ℓ/∂σ² = 0 → σ² 추정
∂ℓ/∂σ² = −n/(2σ²) + Σ(xᵢ − μ̂)²/(2σ⁴) = 0 ⟹ σ̂²_MLE = (1/n) Σ(xᵢ − x̄)²
표본 분산이 MLE 추정량 (분모 n)
⚠ 표본 분산의 분모 n vs n−1
MLE 분산 추정량 (1/n)은 모분산을 과소 추정하는 편향(bias) 추정량이다.
불편 추정량은 (1/(n−1))을 사용한다 (Bessel 보정).
※ 이유: μ̂(대신 x̄)을 사용하면 자유도가 1 소모되어 n−1로 나눠야 E[σ̂²] = σ² (불편성) 성립
03
최대 사후 확률 추정 (MAP)
베이즈 정리

MLE는 데이터만 본다. MAP는 '사전 지식(Prior)'도 함께 반영한다.

P(θ | X) = P(X | θ) · P(θ) / P(X)
→ Posterior = Likelihood × Prior / Evidence
사전 확률 (Prior)
P(θ)

데이터 관측 전
θ에 대한 사전 믿음

예: 가중치가 0 근처일 것

우도 (Likelihood)
P(X|θ)

θ가 주어졌을 때
데이터가 나올 확률

→ MLE의 목적함수

증거 (Evidence)
P(X)

= ∫ P(X|θ)P(θ)dθ
정규화 상수

θ에 독립적 → 최적화 무관

사후 확률 (Posterior)
P(θ|X)

데이터 관측 후
갱신된 θ의 믿음

→ MAP의 목적함수

MAP 추정량 θ̂_MAP = argmax_θ P(θ|X) = argmax_θ [log P(X|θ) + log P(θ)] ← P(X)는 θ에 무관, log 취해도 argmax 불변
MAP 수식 유도 — Gaussian Prior 적용
설정: n개 데이터가 N(μ, 1)에서 샘플링 (분산 = 1로 알려짐). 파라미터 μ의 사전 분포: μ ~ N(0, τ²)
①
로그 우도
log P(X|μ) = −½ Σ(xᵢ − μ)² + const
②
로그 사전 확률
log P(μ) = −μ²/(2τ²) + const
③
MAP 목적함수
Q(μ) = −½ Σ(xᵢ − μ)² − μ²/(2τ²) → 미분 후 0으로 놓으면:
④
최적 μ̂_MAP
μ̂_MAP = nτ² / (1 + nτ²) · x̄ ← 사전 평균(0)과 표본 평균(x̄)의 가중 평균!
MLE vs MAP 비교
비교 항목 MLE MAP
목적함수 argmax Σ log p(xᵢ|θ) argmax [Σ log p(xᵢ|θ) + log P(θ)]
Prior 사용 ✗ 사용 안 함 ✓ Prior P(θ) 반영
데이터 적을 때 불안정 (과적합 위험) Prior가 안전장치 역할
Prior → 0 이면 — MAP = MLE (Prior 무관)
정규화 연결 없음 Gaussian Prior → L2 / Laplace Prior → L1
MAP = 정규화 (Regularization)의 수학적 근거
Prior P(θ) log P(θ) MAP 목적함수 동치인 정규화
Gaussian N(0, 1/λ) −λ/2 · ‖θ‖² Σ log p(xᵢ|θ) − λ‖θ‖² L2 정규화 (Ridge)
Laplace L(0, 1/λ) −λ · ‖θ‖₁ Σ log p(xᵢ|θ) − λ‖θ‖₁ L1 정규화 (Lasso)
04
Single Variable Gaussian 결정 경계
결정 경계 (Decision Boundary) 개념

결정 경계란 클래스를 분류하는 기준점 / 기준면이다.

설정: 클래스 C₁, C₂가 있을 때, 새로운 데이터 x를 어느 클래스에 할당할지 결정해야 한다.
각 클래스의 데이터 분포를 P(x|Cₖ)로 모델링하면 베이즈 정리에 의해 사후 확률을 계산할 수 있다.

베이즈 결정 규칙 (Bayes Decision Rule) x를 C₁으로 분류 ⟺ P(C₁|x) > P(C₂|x) ⟺ P(x|C₁)P(C₁) > P(x|C₂)P(C₂)
결정 경계 (Decision Boundary) P(C₁|x) = P(C₂|x) ⟺ P(x|C₁)P(C₁) = P(x|C₂)P(C₂) ⟺ log 비율 = 0
등 사전 확률
P(C₁) = P(C₂) = 0.5이면
→ P(x|C₁) = P(x|C₂) 인 지점
로그 우도비
log[P(x|C₁)/P(x|C₂)] = 0
을 만족하는 x가 결정 경계
ML 연결
각 클래스의 분포 파라미터를
MLE로 추정 → 결정 경계 도출
수치 예시 — 선형 결정 경계 (LDA, σ₁² = σ₂²)
설정: C₁ ~ N(2, 1), C₂ ~ N(5, 1)
사전 확률 동일 (P(C₁) = P(C₂) = 0.5)
σ₁² = σ₂² → 선형 결정 경계 x* = (μ₁ + μ₂)/2 = (2 + 5)/2 = 3.5
결론:
x < 3.5 → C₁으로 분류
x > 3.5 → C₂로 분류
x* = 3.5는 두 평균의 정확한 중점 (사전 확률이 동일하기 때문)
MAP 결정 경계 — Prior 반영

MAP 결정 경계는 MLE 결정 경계와 동일한 구조이지만 Prior에 의해 위치가 이동된다.

MAP 결정 규칙 x를 C₁으로 분류 ⟺ log P(x|C₁) + log P(C₁) > log P(x|C₂) + log P(C₂)
MLE 기반 결정 경계
P(x|C₁) = P(x|C₂) 인 지점
(사전 확률 무시 또는 균등 가정)

예: x* = (μ₁ + μ₂)/2 = 3.5
(μ₁=2, μ₂=5, 분산 동일 가정)
MAP 기반 결정 경계
P(x|C₁)P(C₁) = P(x|C₂)P(C₂) 인 지점
(사전 확률 반영)

예: P(C₁)=0.3, P(C₂)=0.7이면
x* = 3.5 − [σ²/(μ₁−μ₂)]·log(0.3/0.7)
= 3.5 + 0.28 ≈ 3.78 (C₂ 쪽으로 이동)
해석: C₂가 더 많이 나타나는 (사전 확률이 높은) 경우, 결정 경계가 C₁ 쪽으로 이동 → x가 더 'C₁스럽게' 보여야 C₁으로 분류된다.
분산이 다를 때 — 이차 결정 경계 (QDA)

분산이 다를 때(σ₁² ≠ σ₂²), 결정 경계는 이차 방정식이 되어 두 개의 교점이 생긴다.

LDA (σ₁² = σ₂²)
→ 선형 결정 경계 (직선/초평면)
파라미터 수가 적어 안정적
QDA (σ₁² ≠ σ₂²)
→ 이차 결정 경계 (곡선/이차 초곡면)
유연하지만 데이터가 많이 필요
핵심: 분산이 같으면 선형(직선), 분산이 다르면 이차(포물선) 결정 경계 — 이것이 LDA vs QDA의 차이
05
다변수로의 확장 (Multivariate)
Multinomial / Categorical 분포 MLE

이진 분류 (Bernoulli)를 K개 클래스로 일반화하면 Categorical 분포가 된다.

로그 우도 (Categorical MLE 유도) log L(π) = Σₖ nₖ · log πₖ (nₖ = 클래스 k가 나온 횟수, n = Σnₖ = 전체 샘플 수)
MLE 결과 (라그랑주 승수법) π̂ₖ = nₖ / n (각 클래스의 관측 비율 = MLE 추정량)
Categorical MLE — Softmax 함수 연결

신경망에서 다중 클래스 분류를 위해 Softmax 출력층을 사용한다.
이것은 Categorical 분포의 파라미터를 신경망으로 모델링하는 것이다.

Softmax 함수 P(y=k | x ; W) = softmax(Wx)ₖ = exp(wₖᵀx) / Σⱼ exp(wⱼᵀx)
Cross-Entropy 손실 L = −(1/n) Σᵢ log P(yᵢ | xᵢ ; W) = −(1/n) Σᵢ Σₖ yᵢₖ · log P(y=k | xᵢ ; W)
MLE ↔ Cross-Entropy 최소화의 동치: Cross-Entropy를 최소화하는 것 = MLE
일반화 정리
이진 분류
Bernoulli
↓
Sigmoid + Binary CE
다중 분류
Categorical
↓
Softmax + Categorical CE
회귀
Gaussian
↓
선형출력 + MSE (= MLE)
Multinomial MAP — Dirichlet Prior

Categorical MLE는 데이터가 적을 때 0 확률이 발생할 수 있다. MAP + Dirichlet Prior로 해결한다.

문제: MLE의 한계
클래스 k가 학습 데이터에서 한 번도 나오지 않으면
π̂ₖ = 0/n = 0
→ 새로운 데이터에 대해 P(x|Cₖ) = 0 발생
→ Zero probability problem
해결: MAP + Dirichlet Prior
Dirichlet(α₁, …, αₖ)를 Prior로 사용
αₖ: 각 클래스에 대한 가상 카운트 (pseudo-count)
→ Laplace 평활화 (αₖ=1이면 모든 클래스에 1 추가)
Dirichlet Prior MAP 추정량 π̂ₖ = (nₖ + αₖ − 1) / (n + Σαₖ − K) Laplace 평활화 (αₖ = 1): π̂ₖ = (nₖ + 1) / (n + K)
실제 활용: Naive Bayes 텍스트 분류에서 단어 등장 확률 추정 시 Laplace 평활화(αₖ=1)가 표준적으로 사용된다.
다변량 Gaussian — 결정 경계 확장

1차원 Gaussian → d차원으로 확장하면 결정 경계가 '초평면' 또는 '이차 초곡면'이 된다.

다변량 Gaussian PDF P(x|Cₖ) = N(x ; μₖ, Σₖ) = (2π)^(−d/2) |Σₖ|^(−½) · exp[−½(x−μₖ)ᵀΣₖ⁻¹(x−μₖ)]
LDA (Σ₁ = Σ₂ = Σ)
로그 우도비에서 이차 항 (xᵀΣ⁻¹x)이 상쇄됨
→ 선형 결정 경계 (초평면)

파라미터 수가 적어 안정적
QDA (Σ₁ ≠ Σ₂)
이차 항이 남음
→ 이차 결정 경계 (이차 초곡면)
타원, 쌍곡면, 포물면 등 다양한 형태

유연하지만 데이터가 많이 필요
핵심: 공분산 행렬의 가정에 따라 결정 경계의 형태가 결정된다.
생성 모델 vs 판별 모델
판별 모델 (Discriminative)

학습 목표: P(y|x) 직접 학습

결정 경계만 학습
→ 분류 정확도 높음
→ 새로운 데이터 생성 불가

예시: Logistic Regression, SVM, 신경망 분류기, Transformer (분류)
생성 모델 (Generative)

학습 목표: P(x,y) = P(x|y)P(y) 결합 분포 학습

데이터 생성 분포 자체를 모델링
→ 새로운 데이터 생성 가능

예시: Naive Bayes, Gaussian Mixture Model, VAE, GAN, Diffusion, LLM
베이즈 정리로 연결 P(y|x) = P(x|y) · P(y) / P(x) 생성 모델의 P(x|y), P(y)를 추정하면 → 판별 확률 P(y|x) 도출 가능

핵심 요약

01 확률론적 추론

  • 데이터 = 노이즈 포함 → 확률 모델 필요
  • P(y|x;θ)로 불확실성 수치화
  • MLE: argmax P(X|θ)
  • MAP: argmax P(θ|X)

02 MLE

  • 우도 L(θ;X) = ∏ P(xᵢ|θ)
  • 로그 우도로 곱 → 합 변환
  • Gaussian: μ̂ = x̄, σ̂² = (1/n)Σ(xᵢ−x̄)²
  • MLE 분산 = 편향 추정 (분모 n)

03 MAP

  • = MLE + log Prior
  • Gaussian Prior → L2 (Ridge)
  • Laplace Prior → L1 (Lasso)
  • 데이터 적을 때 Prior가 안전장치

04 결정 경계

  • P(C₁|x) = P(C₂|x) 인 지점
  • σ₁² = σ₂² → LDA (선형)
  • σ₁² ≠ σ₂² → QDA (이차)
  • MAP 경계: Prior에 의해 위치 이동

05 다변수 확장

  • Categorical MLE: π̂ₖ = nₖ/n
  • Dirichlet Prior → Laplace 평활화
  • Softmax = Categorical 파라미터 신경망
  • 생성 모델 vs 판별 모델

핵심 공식 모음

  • θ̂_MLE = argmax Σ log P(xᵢ|θ)
  • θ̂_MAP = argmax [Σ log P(xᵢ|θ) + log P(θ)]
  • x* (LDA) = (μ₁+μ₂)/2
  • Softmax: exp(wₖᵀx) / Σⱼ exp(wⱼᵀx)
