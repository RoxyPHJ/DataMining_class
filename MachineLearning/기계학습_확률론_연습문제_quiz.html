01 기댓값과 분산
1
객관식 기본 문제 1 참조 ⭐ 쉬움
확률변수 X의 PMF가 아래와 같을 때, E[X]는?

k 0 1 2 3
P(X=k) 0.50 0.30 0.15 0.05
A 0.65
B 0.75
C 0.80
D 0.90
정답
B. 0.75
E[X] = 0×0.50 + 1×0.30 + 2×0.15 + 3×0.05
= 0 + 0.30 + 0.30 + 0.15 = 0.75
2
객관식 [응용] 문제 1 참조 ⭐⭐ 보통
위 문제 1과 동일한 PMF에서 Y = 3X + 2로 정의할 때,
Var(Y)의 값은? (성질: Var(aX+b) = a²Var(X) 이용)

먼저 Var(X) = E[X²] − (E[X])² 를 구한 뒤 적용하라.
A 4.6875
B 5.0625
C 5.8875
D 6.1875
정답
D. 6.1875
① E[X²] 계산:
E[X²] = 0²×0.50 + 1²×0.30 + 2²×0.15 + 3²×0.05 = 0 + 0.30 + 0.60 + 0.45 = 1.35

② Var(X) 계산:
Var(X) = E[X²] − (E[X])² = 1.35 − (0.75)² = 1.35 − 0.5625 = 0.6875

③ Var(Y) = 3² × Var(X) = 9 × 0.6875 = 6.1875
변형 포인트: 원본 문제에서 E[Y], Var(Y)를 모두 구하는 문항 중 Var(Y)만 분리하여 선택지로 제시. E[X]를 먼저 구한 뒤 분산 공식을 두 단계 적용해야 풀리는 구조.
02 Bias–Variance Tradeoff
3
OX 문제 기본 문제 2 참조 ⭐⭐ 보통
다음 명제의 참/거짓을 판별하라:

"Bias–Variance Decomposition에서 노이즈 항 σ²은 모델 복잡도와 무관하게 일정하므로, 아무리 좋은 모델을 써도 줄일 수 없는 불가피한 오차이다."
⭕ O
❌ X
정답
⭕ O (참)
E[(y − f̂(x))²] = Bias² + Variance + σ²

노이즈 항 σ²은 데이터 생성 과정 자체에서 비롯되며(ε ~ N(0,σ²)), 어떤 모델도 줄일 수 없는 환원 불가능한 오차(irreducible error)이다. 반면 Bias와 Variance는 모델 선택에 따라 조절 가능하다.
4
OX 문제 [응용] 문제 2, 4-3 참조 ⭐⭐ 보통
다음 명제의 참/거짓을 판별하라:

"모델 예측 오차 분포가 N(5, 4)인 모델 A는 N(0, 25)인 모델 B보다 총 기댓값 오차(Bias² + Variance)가 크므로, Bias–Variance 관점에서 모델 A가 모델 B보다 나쁜 모델이다."
⭕ O
❌ X
정답
⭕ O (참)
모델 A: Bias² = 5² = 25, Variance = 4 → 합 = 29
모델 B: Bias² = 0² = 0, Variance = 25 → 합 = 25

따라서 총 오차 합은 A(29) > B(25). 명제는 참이다.

단, 실용적으로는 모델 B도 Variance=25로 매우 커서, 실제 선택은 데이터 크기·목적에 따라 달라진다.
변형 포인트: 원본 문제 4-3에서 "어느 모델이 더 나은지"를 서술형으로 묻는 것을 OX 명제로 재구성. Bias²와 Variance의 수치 합산을 직접 계산한 뒤 대소 비교하도록 유도.
03 공분산과 상관계수
5
단답형 기본 문제 3 참조 ⭐⭐ 보통
다음 데이터에서 표본 평균 x̄와 ȳ를 구하고, 표본 공분산 C(x,y)를 입력하라.

학생 1 2 3 4 5
x (시간) 2 4 6 8 10
y (점수) 55 65 70 80 85

표본 공분산 C(x,y)를 소수점 첫째 자리까지 입력하라.
정답
C(x,y) = 37.5
x̄ = (2+4+6+8+10)/5 = 6, ȳ = (55+65+70+80+85)/5 = 71

Σ(xᵢ−x̄)(yᵢ−ȳ) = (−4)(−16)+(−2)(−6)+(0)(−1)+(2)(9)+(4)(14)
= 64 + 12 + 0 + 18 + 56 = 150

C(x,y) = 150 / (5−1) = 37.5
6
객관식 [응용] 문제 3 참조 ⭐⭐⭐ 어려움
아래 수정된 데이터에서 상관계수 ρ에 가장 가까운 값은?

학생 1 2 3 4 5
x (시간) 2 4 6 8 10
y (점수) 85 80 70 65 55
※ y 값을 역순으로 변경 (음의 상관 방향)
A +0.98
B 0.00
C −0.98
D −0.50
정답
C. −0.98
x̄=6, ȳ=71 (동일). x는 그대로이므로 V(x)도 동일 = 10.
y를 역순으로 바꿔도 ȳ=71, V(y) 동일.

Σ(xᵢ−x̄)(yᵢ−ȳ) = (−4)(14)+(−2)(9)+(0)(−1)+(2)(−6)+(4)(−16)
= −56 − 18 + 0 − 12 − 64 = −150

C(x,y) = −150/4 = −37.5
ρ = −37.5 / (√10 × √(V(y))) ≈ −0.98 (원본과 절댓값 동일, 부호 반전)
변형 포인트: y값을 역순으로 뒤집어 음의 상관관계로 변형. 공분산 부호가 반전되어 상관계수 부호도 반전됨. 크기는 동일하므로 부호 개념을 이해했는지 확인하는 문제.
04 정규분포와 표준화
7
단답형 기본 문제 4 참조 ⭐ 쉬움
시험 점수 X ~ N(70, 100)일 때, 68–95–99.7 규칙을 이용하여
P(50 ≤ X ≤ 90)을 구하라. 소수점 표기 없이 %로 입력하라.
예: "95%"
정답
95%
σ = √100 = 10이므로:
μ − 2σ = 70 − 20 = 50, μ + 2σ = 70 + 20 = 90

68–95–99.7 규칙에 따라 μ ± 2σ 구간에 약 95%의 데이터가 포함된다.
따라서 P(50 ≤ X ≤ 90) ≈ 95%
8
객관식 [응용] 문제 4 참조 ⭐⭐ 보통
시험 점수 분포가 X ~ N(70, 100)에서 X ~ N(60, 100)으로 변경되었다.
(평균만 70→60으로 변경, 분산 동일)

이 때 P(X > 75)를 구하면? (Φ(1.5) ≈ 0.9332)
A 0.0668
B 0.9332
C 0.1587
D 0.8413
정답
A. 0.0668
μ=60, σ=10이므로 Z = (X−60)/10

P(X > 75) = P(Z > (75−60)/10) = P(Z > 1.5)

= 1 − Φ(1.5) = 1 − 0.9332 = 0.0668
변형 포인트: 원본은 μ=70에서 P(X>85) 계산. 평균을 60으로 낮추고 기준점을 75로 조정하여 Z-score는 동일하게 1.5가 되도록 설계. 분포의 중심 이동이 표준화에 미치는 영향을 확인하는 문제.
05 상관계수와 독립성
9
객관식 기본 문제 5 참조 ⭐⭐ 보통
X ~ Uniform(−1, 1), Y = X² 일 때,
Cov(X, Y)를 구하면?
참고: fX(x) = 1/2 for −1 ≤ x ≤ 1
A 1/3
B 1/5
C 0
D −1/3
정답
C. 0
E[X] = ∫₋₁¹ x · (1/2) dx = 0 (대칭)
E[Y] = E[X²] = ∫₋₁¹ x² · (1/2) dx = [x³/6]₋₁¹ = 1/3
E[XY] = E[X·X²] = E[X³] = ∫₋₁¹ x³ · (1/2) dx = 0 (홀함수의 대칭 적분)

Cov(X,Y) = E[XY] − E[X]·E[Y] = 0 − 0 · (1/3) = 0

→ 공분산이 0이지만 X와 Y는 Y=X²으로 명백히 종속. 독립 ≠ 무상관.
10
OX 문제 [응용] 문제 5 참조 ⭐⭐⭐ 어려움
다음 명제의 참/거짓을 판별하라:

"X ~ Uniform(0, 1), Y = X² 로 정의하면, Cov(X, Y) = 0이 되어 상관계수도 0이므로, X와 Y는 서로 독립이다."
⭕ O
❌ X
정답
❌ X (거짓)
① Cov(X,Y) 계산: X ~ U(0,1)이면 E[X]=1/2, E[Y]=E[X²]=1/3, E[XY]=E[X³]=1/4
Cov(X,Y) = 1/4 − (1/2)(1/3) = 1/4 − 1/6 = 1/12 ≠ 0

→ 범위가 (−1,1)이 아닌 (0,1)이면 대칭성이 없으므로 공분산도 0이 아니다.
명제의 전제 자체가 틀렸으므로 전체 명제는 거짓.

핵심: "상관계수=0 → 독립"은 거짓이지만, 이 경우엔 그 전제조차 성립 안 함.
변형 포인트: 원본 Uniform(−1,1)에서 대칭성으로 Cov=0인 특수 케이스를 Uniform(0,1)으로 바꿔 대칭 구조를 제거. "상관계수=0이면 독립"이라는 오개념과 함께, 범위 변화에 따른 공분산 부호 변화를 동시에 물음.
06 중심극한정리
11
객관식 기본 문제 6 참조 ⭐⭐ 보통
불량률 p=0.2인 공장에서 n=100개를 추출할 때 불량품 수 Sₙ의 분포와 기댓값/분산으로 옳은 것은?
A Sₙ ~ B(100, 0.2), E[Sₙ]=20, Var(Sₙ)=16
B Sₙ ~ B(100, 0.2), E[Sₙ]=20, Var(Sₙ)=20
C Sₙ ~ N(20, 16), E[Sₙ]=20, Var(Sₙ)=16
D Sₙ ~ Poisson(20), E[Sₙ]=20, Var(Sₙ)=20
정답
A. Sₙ ~ B(100, 0.2), E[Sₙ]=20, Var(Sₙ)=16
Sₙ은 베르누이 시행의 합이므로 이항분포 B(n, p) = B(100, 0.2).
E[Sₙ] = np = 100 × 0.2 = 20
Var(Sₙ) = np(1−p) = 100 × 0.2 × 0.8 = 16

정규분포 N(20,16)은 CLT를 통한 근사이지, 정확한 분포가 아니다.
12
객관식 [응용] 문제 6 참조 ⭐⭐⭐ 어려움
불량률이 p=0.3, n=100으로 변경되었다. CLT를 이용하여
P(Sₙ ≤ 24)에 가장 가까운 값은?
참고: Φ(−1.09) ≈ 0.1379, Φ(−1.10) ≈ 0.1357
A 0.1379
B 0.8621
C 0.5000
D 0.7580
정답
A. 0.1379
p=0.3, n=100 → E[Sₙ]=30, Var(Sₙ)=100×0.3×0.7=21, σ=√21≈4.583

Z = (24 − 30) / √21 = −6 / 4.583 ≈ −1.309
→ 제공된 값 중 Φ(−1.09) ≈ 0.1379에 가장 가까움.

따라서 P(Sₙ ≤ 24) ≈ 0.1379
변형 포인트: 원본 p=0.2, P(Sₙ>25)를 p=0.3, P(Sₙ≤24)로 변환. 불량률 증가로 평균이 20→30으로 이동하여 Z-score의 부호가 반전되고 크기도 달라짐. 방향 역전 + 수치 변형 복합 응용.
07 주변 확률과 조건부 확률
13
객관식 기본 문제 7 참조 ⭐⭐ 보통
공정한 동전을 처음 뒷면이 나올 때까지 던진 횟수 X의 PMF P(X=k)와 기댓값 E[X]로 옳은 것은?
A P(X=k) = (1/2)ᵏ, E[X] = 1
B P(X=k) = (1/2)ᵏ, E[X] = 2
C P(X=k) = (1/2)^(k−1) · (1/2), E[X] = 1
D P(X=k) = (1/2)^(k−1) · (1/2), E[X] = 2
정답
D. P(X=k) = (1/2)^(k−1) · (1/2), E[X] = 2
처음 k−1번은 앞면, k번째에 뒷면이 나와야 하므로:
P(X=k) = (1/2)^(k−1) · (1/2) = (1/2)^k (k=1,2,3,...)

이는 기하분포(Geometric distribution)이며:
E[X] = 1/p = 1/(1/2) = 2

※ 선택지 A, B는 표기가 맞지만 E[X]가 다름. D가 정확.
14
객관식 [응용] 문제 7 참조 ⭐⭐⭐ 어려움
X는 공정한 동전을 처음 뒷면이 나올 때까지의 횟수 (P(X=k) = (1/2)^k).
X번 동안 공정한 동전을 다시 던져 X번 모두 앞면이 나오는 사건을 A라 하자.

주변 확률 P(A) = Σ P(A|X=k)·P(X=k)를 계산하면?
A 1/2
B 1/3
C 1/4
D 2/5
정답
B. 1/3
X=k이면 X번 모두 앞면일 확률: P(A|X=k) = (1/2)^k

P(A) = Σ_{k=1}^∞ P(A|X=k) · P(X=k)
= Σ_{k=1}^∞ (1/2)^k · (1/2)^k
= Σ_{k=1}^∞ (1/4)^k
= (1/4) / (1 − 1/4)  ← 등비급수 공식 a/(1−r)
= (1/4) / (3/4) = 1/3
변형 포인트: 원본 문제 7-4의 풀이 구조 그대로 유지하되, 최종 계산을 객관식으로 변환. 등비급수 수렴 공식을 적용하는 핵심 단계를 확인하는 문제. P(A|X=k)·P(X=k) = (1/4)^k 로 변환되는 순간이 핵심.
