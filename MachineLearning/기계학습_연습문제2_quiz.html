y값 변경 (1,3,2 → 2,2,4), 예측 지점 x=4 → x=5
Step 1 합산값
Σxᵢ = 1+2+3 = 6,   Σyᵢ = 2+2+4 = 8
Σxᵢ² = 1+4+9 = 14,   Σxᵢyᵢ = 1·2+2·2+3·4 = 2+4+12 = 18
Step 2 연립방정식
∂J/∂θ₀=0 :   θ₀·N + θ₁·Σxᵢ = Σyᵢ
   → 3θ₀ + 6θ₁ = 8   … ①
∂J/∂θ₁=0 :   θ₀·Σxᵢ + θ₁·Σxᵢ² = Σxᵢyᵢ

①×2 : 6θ₀ + 12θ₁ = 16
②−①×2 : 2θ₁ = 2  → θ*₁ = 1

Step 4 예측
ŷ(5) = 1·5 + 2/3 = 5 + 2/3
θ*₀ = 2/3,   θ*₁ = 1,   ŷ(5) = 17/3 ≈ 5.67
y값 분포가 바뀌어도 연립방정식 풀이 구조(①×2 → 소거)는 동일함. Σxᵢ, Σxᵢ²는 x가 같으므로 변하지 않는다.
[응용 1-B] 역방향 문제
θ*₁ 역산 ⭐⭐⭐ 어려움
원본 데이터(xᵢ = 1,2,3)를 유지하되, 아래 조건을 만족시키고자 한다.
ŷ(x=4) = 3    (즉, 4에서의 예측값이 정확히 3)
  1. θ*₁과 θ*₀ 사이의 관계식 θ*₀ = 3 − 4θ*₁ 을 도출하라.
  2. 원본 문제의 연립방정식 구조를 활용하여, 이 조건을 만족하는 yᵢ 값의 조합 하나를 역으로 제시하라. (Σyᵢ 와 Σxᵢyᵢ 를 먼저 결정하라.)
  3. 원본 데이터(yᵢ = 1,3,2)와 비교했을 때, ŷ(4)가 달라지는 이유를 θ*₁의 변화 관점에서 설명하라.
결과값(ŷ=3) 고정 → θ*, yᵢ 역산. 풀이 방향이 순방향과 반대.
Step 1 조건 설정
ŷ(4) = 4θ*₁ + θ*₀ = 3  → θ*₀ = 3 − 4θ*₁   … (*)
Step 2 역산
xᵢ = 1,2,3 고정이므로 Σxᵢ=6, Σxᵢ²=14는 변하지 않는다.
연립방정식 구조 :
  ① 3θ₀ + 6θ₁ = Σyᵢ
  ② 6θ₀ + 14θ₁ = Σxᵢyᵢ
(*)를 ①에 대입: 3(3−4θ₁) + 6θ₁ = Σyᵢ → 9−6θ₁ = Σyᵢ
θ₁을 자유롭게 선택. 예) θ*₁ = 1 → θ*₀ = −1, Σyᵢ = 3
→ yᵢ 예시: (0, 1, 2) — 이를 검증하면 Σxᵢyᵢ = 0+2+6=8 일치.
Step 3 비교 설명
원본 데이터의 최적 기울기 θ*₁ = 1/2이었으나,
ŷ(4)=3이 되려면 θ*₁=1이 되어야 하므로 기울기가 커져야 하며
이는 y값의 증가 폭이 더 가파른 데이터로 변경해야 가능하다.
yᵢ = (0,1,2) 등 Σyᵢ=3, Σxᵢyᵢ=8 을 만족하는 조합
→ θ*₀=−1, θ*₁=1, ŷ(4)=3 ✓
역방향 문제 핵심: 결과(ŷ)→θ 조건→(Σyᵢ, Σxᵢyᵢ)→yᵢ 역산 순서로 접근.

θ₀⁽¹⁾=0, θ₁⁽¹⁾=0.25, P(y=1|x=1) = σ(0.25) ≈ 0.562

[응용 2-B] 초기 파라미터 변형

  4. 1스텝 경사하강법 후 θ₀⁽¹⁾, θ₁⁽¹⁾을 구하라.
  5. 원본(θ₀=0 시작)과 비교하여 θ₁ 업데이트 방향이 왜 동일한지 설명하라.
초기값 θ₀=0→0.5 변경. zᵢ≠0이 되어 ŷᵢ가 모두 달라짐.
Step 1 초기 예측
zᵢ = 0.5 + 0·xᵢ = 0.5 (모든 i)
ŷᵢ = σ(0.5) ≈ 0.622 (모든 i)

i=1(y=0): 0.622−0 = 0.622
i=2(y=0): 0.622
i=3(y=1): 0.622−1 = −0.378
i=4(y=1): −0.378

∂J/∂θ₁ = (1/4)[0·0.622 + 1·0.622 + 2·(−0.378) + 3·(−0.378)]
= (1/4)[0 + 0.622 − 0.756 − 1.134] = (1/4)(−1.268) = −0.317
Step 4 업데이트 (α=1)
θ₀⁽¹⁾ = 0.5 − 0.122 = 0.378

가우시안 분류 — 결정 경계

Gaussian Classification · MLE · Decision Boundary

원본 문제 요약
문제 3 ⭐⭐ 보통
C1={1,2,3}, C2={5,6,7}, 동일 분산 σ².
MLE로 µ₁, µ₂, σ² 추정 후 결정 경계 x* 계산.
P(C1)=P(C2)=1/2 (등사전확률)
x* = (µ₁ + µ₂)/2   (등사전확률, 동일분산 조건)
[응용 3-A] 수치 변형
데이터 스케일 2배 이동 ⭐⭐ 보통
두 클래스가 아래와 같이 주어졌다. 동일 분산 가정, P(C1)=P(C2)=1/2.
클래스데이터nₖ
C1{2, 4, 6}3
C2{8, 10, 12}3
  1. MLE로 µ̂₁, µ̂₂ 를 구하라.
  2. 공유 분산 σ̂² 를 계산하라.
  3. 로그 우도비를 전개하여 결정 경계가 x에 대한 일차식임을 보여라.
  4. 결정 경계 x* 를 구하라.
  5. x=6과 x=9를 분류하라.
C1={1,2,3}→{2,4,6}, C2={5,6,7}→{8,10,12} (스케일 2배), 예측 지점 변경
Step 1 평균
µ̂₁ = (2+4+6)/3 = 4,   µ̂₂ = (8+10+12)/3 = 10
Step 2 공유 분산
C1 편차²: (2−4)²+(4−4)²+(6−4)² = 4+0+4 = 8
C2 편차²: (8−10)²+(10−10)²+(12−10)² = 4+0+4 = 8
σ̂² = (8+8)/(3+3) = 16/6 = 8/3 ≈ 2.667
Step 3 로그 우도비
log[P(x|C1)/P(x|C2)] = [−(x−µ₁)²+（x−µ₂)²] / (2σ²)
= [−(x−4)²+(x−10)²] / (2σ²)
분자: −(x²−8x+16)+(x²−20x+100) = 12x−116 + (84) → 12x − 84
※ x² 소거 확인 ✓
→ (12x−84)/(2σ²)   (x의 일차식)
Step 4 결정 경계
log 우도비 = 0 → 12x−84 = 0 → x* = 7
(= (µ₁+µ₂)/2 = (4+10)/2 = 7 ✓)
Step 5 분류
x=6 : 6 < 7 → C1
x=9 : 9 > 7 → C2
µ̂₁=4, µ̂₂=10, σ̂²=8/3, x*=7  |  x=6→C1, x=9→C2
원본 대비 모든 수치가 2배 스케일이지만 결정 경계는 여전히 (µ₁+µ₂)/2. 분산도 4배(원본 4/3→8/3)로 커지지만 등사전확률·동일분산 조건에서 결정 경계 위치엔 분산값이 영향 없음.
[응용 3-B] 조건 변형 — 비대칭 사전확률
P(C1)≠P(C2) 조건 ⭐⭐⭐ 어려움
원본 데이터(C1={1,2,3}, C2={5,6,7})를 그대로 사용하되,
사전 확률을 P(C1)=1/3, P(C2)=2/3 으로 변경한다.
  1. 원본에서 구한 µ̂₁=2, µ̂₂=6, σ̂²=4/3 을 사용하여 로그 우도비를 전개하라.
  2. 등사전확률이 아닐 때 결정 경계 방정식은 다음과 같다:
    log P(x|C1) + log P(C1) = log P(x|C2) + log P(C2)
    log(1/3)과 log(2/3) 항을 포함하여 결정 경계 x*를 구하라.
  3. 원본 x*=4와 비교하여 결정 경계가 어느 방향으로 이동했는지 설명하라.
  4. x=3.5를 어느 클래스로 분류하는가? 원본 분류 결과와 다른가?
P(C1)=1/2→1/3, P(C2)=1/2→2/3 (비대칭 사전확률). 결정 경계 이동 분석.
Step 1 로그 우도비 전개
µ₁=2, µ₂=6, σ²=4/3
log[P(x|C1)/P(x|C2)] = [−(x−2)²+(x−6)²]/(2·4/3)
분자: −x²+4x−4+x²−12x+36 = −8x+32
= (−8x+32)/(8/3) = 3(−8x+32)/8 = (−3x+12)·1 → 비례식으로 부호만 중요
정리: log 우도비 ∝ −8x+32 (x의 감소 함수)
Step 2 비등사전확률 결정 경계
log P(x|C1) + log P(C1) = log P(x|C2) + log P(C2)
→ log[P(x|C1)/P(x|C2)] = log[P(C2)/P(C1)]
→ (−8x+32)/(8/3) = log(2/3 ÷ 1/3) = log 2 ≈ 0.693
→ (−8x+32)·(3/8) = 0.693
→ −3x+12 = 0.693
→ x* = (12−0.693)/3 ≈ 3.769
Step 3 이동 방향
원본 x*=4 → 새 x*≈3.769: C1 방향(왼쪽)으로 이동.
P(C2)가 더 크므로 C2 영역이 넓어져 경계가 C1 쪽으로 당겨진다.
Step 4 x=3.5 분류
3.5 < 3.769 = x* → C1
원본(x*=4)에서도 3.5 < 4 → C1 → 분류 결과 동일.
(단, x∈(3.769, 4) 범위에서는 분류 결과가 달라짐)
x* ≈ 3.769 (원본 4보다 C1 방향 이동)  |  x=3.5 → C1 (원본과 동일)
P(C2)>P(C1)이면 경계가 C1 쪽으로 이동 → C1 영역이 좁아짐. 원본 대비 (3.769, 4) 구간의 포인트는 C1→C2로 재분류됨.

[응용 4-A] 4클래스 확장
클래스 수 3→4 확장 ⭐⭐⭐ 어려움
4클래스 A,B,C,D, 총 n=10 관측값.
클래스ABCD
관측 횟수 nₖ3322
ℓ = 3log πA + 3log πB + 2log πC + 2log πD
  1. 제약 조건 없이 πA만으로 ∂ℓ/∂πA=0 을 풀면 어떤 결과가 나오는가? 왜 의미 없는가?
  2. 라그랑주 함수 L = ℓ − λ(πA+πB+πC+πD−1) 을 정의하고, ∂L/∂πₖ=0 (k=A,B,C,D) 조건을 써라.
  3. 각 πₖ를 λ의 식으로 표현하라.
  4. 제약 조건에 대입하여 λ를 구하고, πˆₖ를 계산하라.
  5. 결과가 nₖ/n과 일치함을 확인하라.
클래스 수 3→4 확장 (D 추가), 관측 분포 변경 (4,3,3)→(3,3,2,2)
Step 1 제약 없는 최대화
∂ℓ/∂πA = 3/πA = 0 → 해 없음 (πA→∞ 로 발산)
확률이 1 이상이 될 수 있어 의미 없음.
Step 2 라그랑주 함수
L = 3log πA + 3log πB + 2log πC + 2log πD − λ(πA+πB+πC+πD−1)
∂L/∂πA = 3/πA − λ = 0
∂L/∂πB = 3/πB − λ = 0
∂L/∂πC = 2/πC − λ = 0
∂L/∂πD = 2/πD − λ = 0
Step 3 λ로 표현
πA = 3/λ,   πB = 3/λ,   πC = 2/λ,   πD = 2/λ
Step 4 λ 결정 및 πˆₖ
πA+πB+πC+πD = (3+3+2+2)/λ = 10/λ = 1
→ λ = 10
πˆA = 3/10,   πˆB = 3/10,   πˆC = 2/10,   πˆD = 2/10
Step 5 확인
πˆₖ = nₖ/n: 3/10, 3/10, 2/10, 2/10 ✓
λ = n = 10,   πˆA=πˆB=0.3,   πˆC=πˆD=0.2
클래스가 4개로 늘어도 λ=Σnₖ=n임이 동일하게 성립. 라그랑주 승수법의 결론(πˆₖ=nₖ/n)은 클래스 수에 무관한 일반 결과임을 확인할 수 있다.
[응용 4-B] 역방향 문제
πˆA → nA 역산 ⭐⭐⭐ 어려움
3클래스 A,B,C, 총 n=10. 아래 조건이 주어진다.
πˆA = 0.5,   nB = nC (B와 C는 동수)
  1. πˆA = nA/n = 0.5 임을 이용하여 nA를 구하라.
  2. n=10에서 nA를 빼고 nB=nC 조건으로 nB, nC를 구하라.
  3. 로그 우도 함수 ℓ 을 명시하라.
  4. 라그랑주 승수법을 적용하여 πˆB, πˆC 를 구하라.
  5. ∇ℓ = λ·∇g 조건에서 πˆA의 그래디언트 크기(nA/πˆA)와 πˆB의 그래디언트 크기를 비교하여 nA>nB이면 어떤 의미인지 서술하라.
결과(πˆA=0.5) 고정 → nA 역산 후 나머지 분포 결정. 역방향 + 개념 결합.
Step 1 nA 결정
πˆA = nA/10 = 0.5 → nA = 5
Step 2 nB, nC 결정
nB + nC = 10 − 5 = 5,   nB = nC
→ nB = nC = 2.5
※ 실제 횟수는 정수여야 하므로 이 조건을 정확히 만족하는 정수해는 없다.
이론적으로는 nB = nC = 2.5로 처리하거나, n=12(nA=6, nB=nC=3)로 스케일 변환 가능.
Step 3 로그 우도
ℓ = 5log πA + 2.5log πB + 2.5log πC (이론값)
또는 n=12 기준: ℓ = 6log πA + 3log πB + 3log πC
Step 4 πˆB, πˆC
λ = n = 10 (또는 12)
πˆB = nB/n = 2.5/10 = 0.25 (이론), 또는 3/12 = 0.25
πˆC = 0.25 (동일)
Step 5 그래디언트 해석
∂ℓ/∂πˆA = nA/πˆA = 5/0.5 = 10
∂ℓ/∂πˆB = nB/πˆB = 2.5/0.25 = 10
모든 클래스에서 nₖ/πˆₖ = λ = n 으로 동일하다.
이것이 최적 조건(∇ℓ = λ∇g)의 본질 — 모든 클래스의 한계 기여가 동등해지는 점에서 최대화됨.
nA=5, nB=nC=2.5, πˆA=0.5, πˆB=πˆC=0.25
역방향 핵심: πˆₖ=nₖ/n 관계를 역으로 사용하여 nₖ를 역산. 정수 제약 위반은 이론과 현실의 차이를 논의하는 지점으로 활용 가능.
